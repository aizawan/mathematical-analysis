ベルヌーイ型微分方程式 は，次の形式で表される一階微分方程式である．

dydx+p(x)y=q(x)yn\frac{dy}{dx} + p(x)y = q(x)y^ndxdy​+p(x)y=q(x)yn
(1)

ここで，p(x)p(x)p(x) と q(x)q(x)q(x) は考えている区間で定義された xxx の関数であり，nnn は定数である．n≠0,1n\neq 0,1n=0,1 の場合は yny^nyn の項があるため一階非線形微分方程式であるが，適切な変数変換によって一階線形微分方程式に帰着できる．

ベルヌーイ型微分方程式の解法¶

ベルヌーイ型微分方程式は，一階非線形微分方程式の中でも特別な形を持つ．nnn の値によって既に学んだ線形微分方程式になる場合と，変数変換が必要な場合に分かれる．

n=0n=0n=0 の場合¶

まず n=0n=0n=0 の場合を考える．代入すると，

dydx+p(x)y=q(x)\frac{dy}{dx} + p(x)y = q(x)dxdy​+p(x)y=q(x)
(2)

となる．これは一階線形非同次微分方程式であり，積分因子法や定数変化法で解くことができる．

n=1n=1n=1 の場合¶

次に n=1n=1n=1 の場合を考える．代入すると，

dydx+p(x)y=q(x)ydydx+(p(x)−q(x))y=0\begin{aligned} \frac{dy}{dx} + p(x)y &= q(x)y \\ \frac{dy}{dx} + \left(p(x) - q(x) \right)y &= 0 \end{aligned}dxdy​+p(x)ydxdy​+(p(x)−q(x))y​=q(x)y=0​
(3)

となる．これは一階線形同次微分方程式であり，変数分離法で解くことができる．

n≠0,1n\neq0,1n=0,1 の場合¶

ここから n≠0,1n\neq0,1n=0,1 の場合を扱う．この場合は，y≠0y\neq 0y=0 の区間で

(4)

とおく．y,zy,zy,z はどちらも xxx の関数なので，xxx について微分すると，

dzdx=ddxy1−n=(1−n)y−ndydx\begin{aligned} \frac{dz}{dx} &= \frac{d}{dx}y^{1-n} \\ &= \left(1-n \right)y^{-n} \frac{dy}{dx} \end{aligned}dxdz​​=dxd​y1−n=(1−n)y−ndxdy​​

となる．これを使って元の微分方程式を書き換える．まず両辺を yny^nyn で割ると，

dydx+p(x)y=q(x)yn1yndydx+p(x)1yn−1=q(x)\begin{aligned} \frac{dy}{dx} + p(x)y &= q(x)y^n \\ \frac{1}{y^n}\frac{dy}{dx} + p(x) \frac{1}{y^{n-1}} &= q(x) \end{aligned}dxdy​+p(x)yyn1​dxdy​+p(x)yn−11​​=q(x)yn=q(x)​
(6)

である．ここで z=y1−nz=y^{1-n}z=y1−n と dzdx=(1−n)y−ndydx\frac{dz}{dx}=(1-n)y^{-n}\frac{dy}{dx}dxdz​=(1−n)y−ndxdy​ を代入すると，

となる．これは zzz に関する一階線形非同次微分方程式である．積分因子法を用いると，

Pythonによる実装¶

では，上記の微分方程式を SymPy で解く．SymPy の出力は，手計算で得た非零解の分母分子に exe^xex をかけた形になっている．

from sympy import symbols, Eq, Derivative, Function, dsolve

# 変数と未知関数の定義
x = symbols('x')
y = Function('y')(x)

# dy/dxの定義
dy = Derivative(y, x)

# 微分方程式の定義
eq = Eq(dy - y, -x*y**2)

# 微分方程式を解く
y_ = dsolve(eq, y)

黒線は，変数変換の途中で除外された零解 y=0y=0y=0 である．その他の曲線は，分母が 0 になる点で定義されないため，グラフ上では垂直方向に発散することがある．

NumPy + Matplotlib による可視化¶

SymPy の plotting.plot は数式を直接描画できるため便利であるが，分母が 0 になる点の近くでは曲線が不自然につながって見えることがある．

そこで，SymPy の式を NumPy で評価できる関数に変換する lambdify を使い，定義されない点の近くや表示範囲を大きく外れる点を nan として除外して描画する．Matplotlib は nan を含む箇所では線をつながないため，発散点の前後を別の曲線として表示できる．

from sympy import lambdify

y_func = lambdify((x, C1), y_.rhs, 'numpy')
den_func = lambdify((x, C1), 1 / y_.rhs, 'numpy')

xs = np.linspace(0, 5, 3000)
ylim = (-5, 20)

fig, ax = plt.subplots()

# 零解
ax.plot(xs, np.zeros_like(xs), color='black')

cm = plt.get_cmap('magma', N)
for i, c in enumerate(np.linspace(-5, 5, N)):
    ys = y_func(xs, c)
    den = den_func(xs, c)
    ys = np.asarray(ys, dtype=float)

    # 発散点や表示範囲を大きく外れる点では線を切る
    ys = np.where(np.isfinite(ys), ys, np.nan)
    ys = np.where(np.abs(den) > 1e-2, ys, np.nan)
    ys = np.where((ylim[0] <= ys) & (ys <= ylim[1]), ys, np.nan)

    ax.plot(xs, ys, color=cm(i))

ax.set_xlim(0, 5)
ax.set_ylim(*ylim)
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
plt.show()
