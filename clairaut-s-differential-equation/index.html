クレロー型微分方程式

Google Colab で開く

クレロー型微分方程式（Clairaut’s equation） は以下の形式を持つ微分方程式である

y=dydxx+f(dydx)y=\frac{dy}{dx}x + f \left( \frac{dy}{dx}\right)y=dxdy​x+f(dxdy​)
(1)

ここで関数 fff はdydx\frac{dy}{dx}dxdy​ の関数である．

クレロー型微分方程式の解法¶

クレロー型微分方程式の解は関数 fff の形に依存するが，導関数 dydx\frac{dy}{dx}dxdy​ について解くまでの一般解の導出を行う

まず導関数 dydx\frac{dy}{dx}dxdy​ を別の関数 ppp に置き換える．式は

y=px+f(p)y=px + f \left(p\right)y=px+f(p)
(2)

となる．続いて，上式をxxxで微分する．y,py,py,p ともにxxxの関数であることに注意すると

dydx=p+xdpdx+f′(p)dpdxp=p+dpdx(x+f′(p))0=dpdx(x+f′(p))\begin{aligned} \frac{dy}{dx}&=p+x\frac{dp}{dx} + f'(p) \frac{dp}{dx} \\ p&=p+ \frac{dp}{dx} \left ( x + f'(p)\right ) \\ 0 &= \frac{dp}{dx} \left ( x + f'(p)\right ) \end{aligned}dxdy​p0​=p+xdxdp​+f′(p)dxdp​=p+dxdp​(x+f′(p))=dxdp​(x+f′(p))​
(3)

となる．第三式が成り立つためには，

dpdx=d2ydx2=0\frac{dp}{dx} = \frac{d^2y}{dx^2} = 0dxdp​=dx2d2y​=0
(4)

もしくは

x+f′(p)=0x+f'(p)=0x+f′(p)=0
(5)

である必要がある．前者の条件は2階微分すると0ということは導関数は定数，つまり

dydx=p=C\frac{dy}{dx} = p = Cdxdy​=p=C
(6)

であるため，一般解は

y=Cx+f(C)y=Cx+f(C)y=Cx+f(C)
(7)

となる．一方，後者の条件については，以下の方程式より，pppを消去することで CCC を含まない特殊解を導出できる．

{x=−f′(p)y=px+f(p)\begin{aligned} &\left\{\begin{array}{l} x = -f'(p) \\ y = px+f(p) \end{array}\right.\\ \end{aligned}​{x=−f′(p)y=px+f(p)​​
(8)

ただし，関数 fff の形に応じては解を求めることができないので注意されたい．

クレロー型微分方程式の具体例¶

次の微分方程式を解いてみよう．

y=dydxx+(dydx)2y=\frac{dy}{dx}x + \left( \frac{dy}{dx}\right)^2y=dxdy​x+(dxdy​)2
(9)

解答はクリックで確認できる．

解答

この微分方程式は，クレロー型の微分方程式であるので p=dydxp=\frac{dy}{dx}p=dxdy​として式を変形すると，

y=px+p2y=px + p^2y=px+p2
(10)

このとき，yyy と ppp が xxx の関数であることに注意して両辺を xxx を微分する．

dydx=p+xdpdx+2pdpdxp=p+dpdx(x+2p)0=dpdx(x+2p)\begin{aligned} \frac{dy}{dx}&=p+x\frac{dp}{dx} + 2p \frac{dp}{dx} \\ p&=p+ \frac{dp}{dx} \left ( x + 2p\right ) \\ 0 &= \frac{dp}{dx} \left ( x + 2p\right ) \end{aligned}dxdy​p0​=p+xdxdp​+2pdxdp​=p+dxdp​(x+2p)=dxdp​(x+2p)​
(11)

この等式が成立するためには，

dpdx=d2ydx2=0\frac{dp}{dx} = \frac{d^2y}{dx^2} = 0dxdp​=dx2d2y​=0
(12)

もしくは

x+2p=0x+2p=0x+2p=0
(13)

である必要がある．前者の条件は2階微分すると0ということは導関数は定数，つまり

dydx=p=C\frac{dy}{dx} = p = Cdxdy​=p=C
(14)

であるため，一般解は

y=Cx+C2y=Cx+C^2y=Cx+C2
(15)

となる．後者の条件は，p=−x/2p=-x/2p=−x/2より，式に代入すると，

y=(−x2)x+(−x2)2=−14x2\begin{aligned} y &= \left ( -\frac{x}{2} \right ) x + \left ( -\frac{x}{2} \right )^2 \\ &= - \frac{1}{4} x^2 \end{aligned}y​=(−2x​)x+(−2x​)2=−41​x2​
(16)

という特異解が得られる．

特異解

特異解 とは一般解の任意定数 CCC にどのような値を取っても得られない解のことを言う．特殊解は一般解の任意定数を初期条件で固定した解であり，混同しないように注意されたい．

そして，この特異解は得られた曲線に全てに接している曲線である 包絡線 となっている．

包絡線

包絡線（envelope） とは α\alphaα をパラメータとする微分可能な関数 F(x,y,α)F(x,y,\alpha)F(x,y,α) が与えられたとき，この関数に接して、かつ接線が軌跡となっている曲線である．ほうらくせんと読む．

Pythonによる実装¶

では，上記の微分方程式をsympyで解く．ただし，クレロー型であっても微分方程式の形によってはsympyでは計算できない場合があることに注意されたい．

from sympy import symbols, Eq, Derivative, Function, dsolve

# 変数と未知関数の定義
x = symbols('x')
y = Function('y')(x)

# dy/dxの定義
dy = Derivative(y, x)

# 微分方程式の定義
eq = Eq(y, dy*x + dy**2)

# 微分方程式を解く
y_ = dsolve(eq, y)
