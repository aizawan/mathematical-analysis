Sympyの基礎¶

Sympyは数式や変数を記号のまま代数的に計算できるPythonのライブラリである．数理解析で学ぶ微分方程式・フーリエ解析をSympyで理解するための最小限の使い方をここでまとめる．

まず，変数（variable） とは，任意の範囲で任意の値を取ることができる記号（文字）である．具体的には，y=2x+1y=2x+1y=2x+1という一次関数があったとき，変数は xxx と yyy である．

Pythonにおける変数と計算の扱いを確認する．

x = 1
y = 2 * x + 1

print(f'x={x}, y={y}')
x=1, y=3

y=2x+1y=2x+1y=2x+1 という式に x=1x=1x=1 を代入したときの結果を計算できた．

Caution

プログラミングにおける変数と数式の変数は意味が異なる点に注意されたい．以下のようにプログラミングでは，変数に値を代入し，値をメモリ上に確保している．データの型に応じて任意の値を代入可能であるが，y=2x+1y=2x+1y=2x+1という数式に現れる変数とは違い，すでに値を代入して式として扱っている点が異なる．

例えば，[−1,1][-1,1][−1,1] の範囲で，y=2x+1y=2x+1y=2x+1 を描画したいとき，この範囲の値を各代入する必要がある．

numpy を使って次のようにプログラミングする．

import numpy as np
x = np.linspace(-1, 1, 100)
y = 2 * x + 1

上記のセルでは，np.linspace を使って，[−1,1][-1,1][−1,1] の範囲に 100 個の点からなる配列を作成し，この配列を y の式に代入している．

そのため，以下のように .shape で確認すると，x と y は複数の値から構成されていることがわかる．

print(x.shape)
print(y.shape)
(100,)
(100,)

matplotlib を使って可視化しよう．

plt.plot(x, y, label='y = 2x + 1')
plt.xlabel('x')
plt.ylabel('y')
plt.grid(True)
plt.legend()
plt.show()

本講義で扱うSympyでは，代入をすることなく，文字として変数を扱ったプログラミングが可能である．つまり，これまでの数学の授業で扱ってきたように数式と変数を定義できる．まずはSympyから変数を扱うためのsymbolsをimportする．

関数¶

関数（function） とは，ある変数 xxx について，一つの変数 yyy に対応させるものである．

y=f(x)y = f(x)y=f(x)
(1)

ここで f(⋅)f(\cdot)f(⋅) を関数と呼び，変数 xxx が取りうる範囲を 定義域（domain） ，変数 yyy が取りうる値を 値域（range） と言う．

具体的には以下のようなものを関数と呼ぶ．

y=2x+1,y=x2−1,y=sin⁡x\begin{align} &y = 2x + 1, \\ &y = x^2 - 1, \\ &y = \sin x \end{align}​y=2x+1,y=x2−1,y=sinx​​

他にもこれまで様々な関数と出会ってきたはずである．また前述したような関数は変数を一つだけ取りうるため一変数関数と呼ばれ，多変数を受け取る多変数関数も考えることができる（後述）．

以降で，Sympyを使って変数を定義して，関数を定義することを試みる．

微積分¶

本講義で扱う微分方程式に 微分 という言葉が含まれていることからもわかるように，微分は関数の性質を知る上で非常に重要な役割を持つ．まずは一変数関数から微分の意味を復習する．

微分¶

これまでの中学〜大学の講義で微分とは関数の傾き・変化量であると習ったはずである．この関数の傾きを知るために，点 x1x_1x1​ で関数 fff に接する直線 ggg を考える．この直線 ggg は接線であった．接線の傾きを知るために，ある二点 x1,x2x_1,x_2x1​,x2​を通る直線の傾きを考える．この傾き aaa は以下の式に示すように増加量で計算することができる．

a=f(x2)−f(x1)x2−x1a=\frac{f\left(x_{2}\right)-f\left(x_{1}\right)}{x_{2}-x_{1}}a=x2​−x1​f(x2​)−f(x1​)​
(5)

x1x_1x1​での傾きを知るために，x2x_2x2​をx1x_1x1​に限りなく近づける．x1x_1x1​をxxx，x2−x1x_2-x_1x2​−x1​をhhhとして極限を使って表すと，

lim⁡h→0f(x+h)−f(x)h\lim _{h \rightarrow 0} \frac{f(x+h)-f(x)}{h}h→0lim​hf(x+h)−f(x)​
(6)

となる．この極限の値は，fffの 導関数（derivative） と呼び，f′(x)f'(x)f′(x) または dfdx\frac{df}{dx}dxdf​と書く．この導関数を求めることを 微分 するという．

またf(n)f^{(n)}f(n)はnnn階の導関数であり，微分方程式に含まれる導関数の最大の次数 nnn を微分方程式の階数と呼ぶ．

sympyではdiffメソッドを利用すると容易に微分が計算できる．関数 y=x2+2x+1y=x^2+2x+1y=x2+2x+1 の一次微分 f′(x)f'(x)f′(x) を計算する．

偏微分¶

多変数関数において，ある変数以外を固定して，その変数で微分することを 偏微分 (partial differentiation) と呼ぶ．変数 x,yx,yx,y を入力として受け取る多変数関数 f(x,y)f(x,y)f(x,y) の偏微分は以下の式で与えられ，偏微分して得られる関数を 偏導関数 (partial derivative) と呼ぶ．

∂f∂x(x,y)=lim⁡h→0f(x+h,y)−f(x,y)h∂f∂y(x,y)=lim⁡h→0f(x,y+h)−f(x,y)h\begin{align} \frac{\partial f}{\partial x}(x, y)&=\lim_{h \rightarrow 0} \frac{f(x+h, y)-f(x, y)}{h} \\ \frac{\partial f}{\partial y}(x, y)&=\lim_{h \rightarrow 0} \frac{f(x, y+h)-f(x, y)}{h} \end{align}∂x∂f​(x,y)∂y∂f​(x,y)​=h→0lim​hf(x+h,y)−f(x,y)​=h→0lim​hf(x,y+h)−f(x,y)​​​
(8)

偏微分を使うと，ある点で曲面に接する平面，接平面(tangent plane) を書くことができる．そのために，勾配 (gradient) ∇f\nabla f∇f を定義する．

∇f=(∂f∂x,∂f∂y)\nabla f = (\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y})∇f=(∂x∂f​,∂y∂f​)
(9)

級数展開¶

級数展開はある関数 fff を関数の組み合わせで近似する手法である．本講義ではフーリエ解析の基礎として三角関数によるフーリエ級数展開を紹介する．その理解のために，関数を無限級数で近似するテイラー展開について復習する．

テイラー展開¶

テイラー展開は，関数を多項式で近似する手法であり以下のように関数 f(x)f(x)f(x) をある点 aaa 周りで表す．Taylor 級数を考えるには関数が十分な回数だけ微分可能である必要がある．また，無限回微分可能であっても Taylor 級数が元の関数に一致するとは限らないため，ここでは有限次数の Taylor 多項式による近似として理解する．

f(x)=f(a)+f′(a)(x−a)+f′′(a)(x−a)22!+f′′′(a)(x−a)33!+⋯f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)(x-a)^2}{2!} + \frac{f'''(a)(x-a)^3}{3!} + \cdotsf(x)=f(a)+f′(a)(x−a)+2!f′′(a)(x−a)2​+3!f′′′(a)(x−a)3​+⋯
(15)

ここで，f′f'f′，f′′f''f′′，f′′′f'''f′′′などはそれぞれ関数fffの1次，2次，3次の導関数を表す．

テイラー展開はSympyでseries関数として実装されており，第一引数には関数を，第二引数に変数を，第三引数にどの点の周りで展開するかを，第四引数に剰余項の次数の上限を与える．以下はf(x)=sin⁡(x)f(x) = \sin(x)f(x)=sin(x)のテイラー展開の実装である（今回は x=0x=0x=0 周りでの展開なので正確にはマクローリン展開である）．

x = symbols('x')
f = sin(x)
