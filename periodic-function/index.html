周期関数

Google Colab で開く

sin⁡x\sin xsinx のグラフは 正弦波 と呼ばれる代表的な波であり，cos⁡x\cos xcosx も sin⁡x\sin xsinx を位相方向にずらした同じ形の周期的な波である．このように同じ波形が一定間隔で繰り返される関数を 周期関数 と呼ぶ．

関数 fff に対して，ある正の定数 T>0T>0T>0 が存在し，すべての xxx について

f(x+T)=f(x)f(x+T) = f(x)f(x+T)=f(x)
(1)

が成り立つとき，fff を周期関数と呼び，このような TTT を 周期 と呼ぶ．sin⁡x\sin xsinx と cos⁡x\cos xcosx は周期 T=2πT=2\piT=2π の周期関数であり，三角関数の性質から，

sin⁡(x+2π)=sin⁡(x),cos⁡(x+2π)=cos⁡(x)\sin(x+2\pi) = \sin(x), \quad \cos(x+2\pi) = \cos(x)sin(x+2π)=sin(x),cos(x+2π)=cos(x)
(2)

が成り立つ．周期のうち最小の正の値が存在するとき，それを 基本周期 と呼ぶ．たとえば sin⁡x\sin xsinx と cos⁡x\cos xcosx の基本周期は 2π2\pi2π である．一方で，2T,3T,...2T, 3T, ...2T,3T,... のような正の整数倍も周期であるが，基本周期とは区別する．なお，定数関数のように任意の正の数が周期となり，基本周期を持たない周期関数もある．

また，sin⁡kx\sin kxsinkx や cos⁡kx\cos kxcoskx の基本周期は 2π/k2\pi/k2π/k である．kkk が大きくなるほど，同じ区間に含まれる波の数が増える．この考え方は，後で扱う周波数・角周波数や，フーリエ級数で関数を正弦波の重ね合わせとして表す考え方につながる．

Pythonによる実装¶

では，sin⁡x\sin xsinx と cos⁡x\cos xcosx について，周期 2π2\pi2π を足しても関数の値が変わらないことを SymPy で確認し，関数をプロットしてみよう．

from sympy import symbols, sin, cos, pi, simplify
from sympy.plotting import plot

x = symbols('x')

sin_periodicity = simplify(sin(x + 2*pi) - sin(x))
cos_periodicity = simplify(cos(x + 2*pi) - cos(x))

print(f"sin(x + 2*pi) - sin(x) = {sin_periodicity}")
print(f"cos(x + 2*pi) - cos(x) = {cos_periodicity}")

plot(
    sin(x), cos(x),
    (x, -3*pi, 3*pi),
);
sin(x + 2*pi) - sin(x) = 0
cos(x + 2*pi) - cos(x) = 0

周期・周波数・角周波数¶

周期 TTT は，波形が 1 回繰り返されるまでの長さを表す．一方，周波数 は単位長さあたりに波形が何回繰り返されるかを表す量である．周波数を ν\nuν とすると，周期と周波数には

ν=1T\nu=\frac{1}{T}ν=T1​
(3)

という関係がある．周期が短いほど，同じ区間に多くの波が入るため，周波数は大きくなる．

sin⁡x\sin xsinx は長さ 2π2\pi2π の区間に 1 個の波を持つ．一方，sin⁡kx\sin kxsinkx は長さ 2π2\pi2π の区間に kkk 個の波を持つ．したがって，kkk は波の細かさを決める係数であり，角度 xxx に対する 角周波数 とみなすことができる．

時間 ttt の関数として波を考える場合，正弦波は

sin⁡(ωt)\sin(\omega t)sin(ωt)
(4)

のように書かれる．ここで，ω\omegaω を 角周波数 と呼ぶ．この波の周期 TTT は

ωT=2π\omega T=2\piωT=2π
(5)

を満たすので，

T=2πω,ν=1T=ω2πT=\frac{2\pi}{\omega}, \qquad \nu=\frac{1}{T}=\frac{\omega}{2\pi}T=ω2π​,ν=T1​=2πω​
(6)

となる．つまり，角周波数と周波数は

ω=2πν\omega=2\pi\nuω=2πν
(7)

で結びついている．フーリエ級数では，基本周期 TTT に対応する基本角周波数 ω0=2π/T\omega_0=2\pi/Tω0​=2π/T を考え，ω0,2ω0,3ω0,…\omega_0, 2\omega_0, 3\omega_0, \ldotsω0​,2ω0​,3ω0​,… の角周波数を持つ三角関数を足し合わせて関数を表す．

plot(
    sin(x),
    sin(2*x),
    sin(3*x),
    (x, -pi, pi),
    legend=True,
);

振幅と位相¶

正弦波は一般に

f(x)=Asin⁡(kx+ϕ)f(x)=A\sin(kx+\phi)f(x)=Asin(kx+ϕ)
(8)

のように書くことができる．ここで，AAA は 振幅，ϕ\phiϕ は 位相 を表す．振幅は波の大きさを決める値であり，A>0A>0A>0 のとき Asin⁡(kx+ϕ)A\sin(kx+\phi)Asin(kx+ϕ) は最大値 AAA，最小値 −A-A−A をとる．一般には振幅は ∣A∣|A|∣A∣ である．

kkk は波の繰り返しの細かさを決める値であり，xxx に対する角周波数に対応する．k>0k>0k>0 のとき，sin⁡(kx)\sin(kx)sin(kx) の基本周期は 2π/k2\pi/k2π/k である．一方，ϕ\phiϕ は波の横方向のずれを表す．sin⁡(kx+ϕ)\sin(kx+\phi)sin(kx+ϕ) は，sin⁡(kx)\sin(kx)sin(kx) を xxx 軸方向に −ϕ/k-\phi/k−ϕ/k だけ平行移動した波である．したがって，ϕ>0\phi>0ϕ>0 のときグラフは左にずれ，ϕ<0\phi<0ϕ<0 のときグラフは右にずれる．

例えば，2sin⁡x2\sin x2sinx は sin⁡x\sin xsinx より振幅が大きい波であり，sin⁡(x+π/4)\sin(x+\pi/4)sin(x+π/4) は sin⁡x\sin xsinx を左に π/4\pi/4π/4 だけずらした波である．

f_base = sin(x)
f_amplitude = 2 * sin(x)
f_phase = sin(x + pi/4)

plot(
    f_base,
    f_amplitude,
    f_phase,
    (x, -2*pi, 2*pi),
    legend=True,

三角関数の合成¶

三角関数を足し合わせることで，より複雑な周期関数を作ることができる．例えば，

g(x)=sin⁡x+12sin⁡2x+13cos⁡3xg(x)=\sin x+\frac{1}{2}\sin 2x+\frac{1}{3}\cos 3xg(x)=sinx+21​sin2x+31​cos3x
(9)

を考える．sin⁡x\sin xsinx，sin⁡2x\sin 2xsin2x，cos⁡3x\cos 3xcos3x はいずれも 2π2\pi2π を周期として持つため，これらを足し合わせた g(x)g(x)g(x) も周期 2π2\pi2π の周期関数である．ここでは基本周期が 2π2\pi2π なので，基本角周波数は ω0=1\omega_0=1ω0​=1 である．したがって，sin⁡x\sin xsinx は ω0\omega_0ω0​，sin⁡2x\sin 2xsin2x は 2ω02\omega_02ω0​，cos⁡3x\cos 3xcos3x は 3ω03\omega_03ω0​ の角周波数を持つ成分と見なせる．このように，基本角周波数の整数倍の三角関数を足し合わせることで，基本周期を保ちながら複雑な波形を表現できる．

g = sin(x) + sin(2*x) / 2 + cos(3*x) / 3

plot(g, (x, -3*pi, 3*pi));

周期的拡張¶

周期関数でない関数であっても，ある区間で定義された形を繰り返すことで周期関数として扱うことができる．この操作を 周期的拡張 と呼ぶ．

例えば，区間 [−π,π)[-\pi, \pi)[−π,π) 上で定義された関数

f(x)=x(−π≤x<π)f(x)=x \qquad (-\pi \leq x < \pi)f(x)=x(−π≤x<π)
(10)

を考える．この関数自体は全実数上では周期関数ではないが，区間 [−π,π)[-\pi, \pi)[−π,π) の形を長さ 2π2\pi2π ごとに繰り返すことで，周期 2π2\pi2π の周期関数を作ることができる．周期的拡張した関数を fper(x)f_{\mathrm{per}}(x)fper​(x) と書くと，

fper(x+2π)=fper(x)f_{\mathrm{per}}(x+2\pi)=f_{\mathrm{per}}(x)fper​(x+2π)=fper​(x)
(11)

が成り立つ．

import numpy as np
import matplotlib.pyplot as plt

# 周期的拡張を表示するため，複数周期にわたる x の値を用意する
x_plot = np.linspace(-4*np.pi, 4*np.pi, 1000)
# 元の関数 f(x)=x を定義する 1 周期分の区間 [-pi, pi) を用意する
# endpoint=False により，次の周期の -pi と重なる pi を含めない
x_base = np.linspace(-np.pi, np.pi, 200, endpoint=False)

# 区間 [-pi, pi) 上では f(x)=x なので，x_base をそのまま関数値とする
f_base = x_base
# x_plot を [-pi, pi) から [0, 2*pi) にずらし，周期 2*pi で割ったあまりをとる．
# すると f_periodic の値は [0, 2*pi) の範囲で同じ形を繰り返す
# そして，最後に -pi を引くことで [-pi, pi) の範囲に戻す
f_periodic = (x_plot + np.pi) % (2*np.pi) - np.pi

plt.figure(figsize=(8, 3))
plt.plot(x_plot, f_periodic, label="periodic extension")
plt.plot(x_base, f_base, linewidth=3, label="f(x)=x on [-pi, pi)")
plt.axhline(0, color="black", linewidth=0.8)
plt.axvline(-np.pi, color="gray", linestyle="--", linewidth=0.8)
plt.axvline(np.pi, color="gray", linestyle="--", linewidth=0.8)
plt.xlim(-4*np.pi, 4*np.pi)
plt.ylim(-np.pi, np.pi)
plt.legend()
plt.show()
