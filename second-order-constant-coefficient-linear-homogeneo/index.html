二階線形微分方程式¶

二階線形微分方程式 は以下のように2階の導関数を持つ微分方程式である．

d2ydx2+p(x)dydx+q(x)y=g(x)\frac{d^2y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = g(x)dx2d2y​+p(x)dxdy​+q(x)y=g(x)
(1)

ここで，p(x)p(x)p(x)，q(x)q(x)q(x)，g(x)g(x)g(x)はxxxの関数である．一階の線形微分方程式で扱ったように，g(x)g(x)g(x)が恒等的に0であるときは二階線形同次微分方程式と呼ばれ，0でないときは二階線形非同次微分方程式と呼ばれる．

二階線形同次微分方程式¶

前述したように二階線形同次微分方程式は以下の形で表される微分方程式である．

d2ydx2+p(x)dydx+q(x)y=0\frac{d^2y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = 0dx2d2y​+p(x)dxdy​+q(x)y=0
(2)

ここで，p(x),q(x)p(x),q(x)p(x),q(x)はxxxの関数である．同次微分方程式であるので右辺が0であることを確認されたい．

解法¶

二階線形同次微分方程式には2つの線形独立な 基本解 y1,y2y_1,y_2y1​,y2​が存在し，これらを重ね合わせた一般解 yyy は以下のようになることが知られている．

y=C1y1+C2y2y=C_1y_1 + C_2y_2y=C1​y1​+C2​y2​
(3)

C1,C2C_1,C_2C1​,C2​は積分定数であり，この一般解は基本解 y1,y2y_1,y_2y1​,y2​ の線形結合であることがわかる．これは微分方程式の線形性から得られる性質（重ね合わせの原理）である．

しかしながら，このような基本解は一般的に求めることができない．ここでは，p(x),q(x)p(x),q(x)p(x),q(x)が定数であるときの二階定数係数線形微分方程式についてのみ紹介する．

重ね合わせの原理の証明

表記を簡略化するために，d2ydx2=y′′\frac{d^2y}{dx^2} = y''dx2d2y​=y′′，dydx=y′\frac{dy}{dx}=y'dxdy​=y′ とすると，二階線形同次微分方程式は

y′′+p(x)y′+q(x)y=0y'' + p(x)y' + q(x)y = 0y′′+p(x)y′+q(x)y=0
(4)

と書ける．ここで，この微分方程式の解空間を

V={y∣y′′+p(x)y′+q(x)y=0}V = \{y | y'' + p(x)y' + q(x)y = 0 \}V={y∣y′′+p(x)y′+q(x)y=0}
(5)

とする．この VVV に含まれる二つの関数 y1,y2y_1, y_2y1​,y2​ と k∈Rk \in \mathbb{R}k∈R について，

u=y1+y2,v=ky1u = y_1 + y_2, \quad v = k y_1u=y1​+y2​,v=ky1​
(6)

の u,vu, vu,v が VVV に含まれることを証明する．

まず，y1,y2y_1, y_2y1​,y2​ はともに微分方程式の解であるので，

y1′′+p(x)y1′+q(x)y1=0y2′′+p(x)y2′+q(x)y2=0\begin{aligned} & y_1''+p(x) y_1'+q(x) y_1=0 \\ & y_2''+p(x) y_2'+q(x) y_2=0 \end{aligned}​y1′′​+p(x)y1′​+q(x)y1​=0y2′′​+p(x)y2′​+q(x)y2​=0​
(7)

である．両辺を足すと

(y1′′+y2′′)+p(x)(y1′+y2′)+q(x)(y1+y2)=0(y1+y2)′′+p(x)(y1+y2)′+q(x)(y1+y2)=0\begin{aligned} \left(y_1''+y_2''\right)+p(x)\left(y_1'+y_2'\right)+q(x)\left(y_1+y_2\right)&=0 \\ \left(y_1+y_2\right)''+p(x)\left(y_1+y_2\right)'+q(x)\left(y_1+y_2\right)&=0 \end{aligned}(y1′′​+y2′′​)+p(x)(y1′​+y2′​)+q(x)(y1​+y2​)(y1​+y2​)′′+p(x)(y1​+y2​)′+q(x)(y1​+y2​)​=0=0​
(8)

ここで，u=y1+y2u=y_1 + y_2u=y1​+y2​ より，

u′′+p(x)u′+q(x)u=0u''+p(x)u'+q(x)u=0u′′+p(x)u′+q(x)u=0
(9)

ゆえに，uuu もまた微分方程式の解となるので u∈Vu \in Vu∈V である．

続いて，任意定数 k∈Rk \in \mathbb{R}k∈R 倍すると，

ky1′′+kp(x)y1′+kq(x)y1=0(ky1)′′+p(x)(ky1)′+q(x)(ky1)=0\begin{aligned} k y_1''+k p(x) y_1'+k q(x) y_1&=0\\ \left(k y_1\right)''+p(x)\left(k y_1\right)'+q(x)\left(k y_1\right)&=0 \end{aligned}ky1′′​+kp(x)y1′​+kq(x)y1​(ky1​)′′+p(x)(ky1​)′+q(x)(ky1​)​=0=0​
(10)

ここで，v=ky1v = ky_1v=ky1​ より，

v′′+p(x)v′+q(x)v=0v''+p(x)v'+q(x)v=0v′′+p(x)v′+q(x)v=0
(11)

ゆえに，vvv もまた微分方程式の解となるので v∈Vv \in Vv∈V である．したがって，重ね合わせの原理が示せる．

微分方程式の解と線形空間

微分方程式の解空間

V={y∣y′′+p(x)y′+q(x)y=0}V = \{y | y'' + p(x)y' + q(x)y = 0 \}V={y∣y′′+p(x)y′+q(x)y=0}
(12)

は前述の関数の和とスカラー倍に関して閉じており，線形空間（ベクトル空間）である．これは解の和とスカラー倍の性質から示される．

さらに，二階線形同次微分方程式の解空間は適切な区間上で2次元になる．したがって，線形独立な2つの解 y1,y2y_1, y_2y1​,y2​ が得られれば，それらは解空間の 基底 となる．一般解の表現

y=C1y1+C2y2y=C_1y_1 + C_2y_2y=C1​y1​+C2​y2​
(13)

は，線形代数の視点でみると，線形空間 VVV の基底 y1,y2y_1, y_2y1​,y2​ による任意の元の表現に対応する．

2つの関数 y1(x),y2(x)y_1(x), y_2(x)y1​(x),y2​(x) の線形独立性を調べるためには，ロンスキー行列式 と呼ばれる行列式を用いる方法がある．

ロンスキー行列式は以下のように定義される

W(y1,y2)(x)=∣y1(x)y2(x)y1′(x)y2′(x)∣=y1(x)y2′(x)−y1′(x)y2(x)W(y_1, y_2)(x) = \begin{vmatrix} y_1(x) & y_2(x) \\ y_1'(x) & y_2'(x) \end{vmatrix} = y_1(x) y_2'(x) - y_1'(x) y_2(x)W(y1​,y2​)(x)=∣∣​y1​(x)y1′​(x)​y2​(x)y2′​(x)​∣∣​=y1​(x)y2′​(x)−y1′​(x)y2​(x)
(14)

ここで，y1,y2y_1,y_2y1​,y2​ が同じ二階線形同次微分方程式の解である場合，W(x)≡0W(x) \equiv 0W(x)≡0 なら y1,y2y_1, y_2y1​,y2​ は線形従属であり，ある点 x0x_0x0​ で W(x0)≠0W(x_0) \neq 0W(x0​)=0 なら y1,y2y_1, y_2y1​,y2​ はその区間で線形独立であるといえる．そして，線形独立であるならば，y1,y2y_1, y_2y1​,y2​ は解空間の基底となり，これを 基本解 という．

関数の線形独立

関数 y1(x),y2(x),…,yn(x)y_1(x), y_2(x), \dots, y_n(x)y1​(x),y2​(x),…,yn​(x) が定義域 III 上で 線形独立 であるとは

C1y1(x)+C2y2(x)+⋯+Cnyn(x)=0(∀x∈I)C_1 y_1(x) + C_2 y_2(x) + \cdots + C_n y_n(x) = 0 \quad (\forall x \in I)C1​y1​(x)+C2​y2​(x)+⋯+Cn​yn​(x)=0(∀x∈I)
(15)

を満たす係数 C1,…,CnC_1, \dots, C_nC1​,…,Cn​ に対して，

C1=C2=⋯=Cn=0C_1 = C_2 = \cdots = C_n = 0C1​=C2​=⋯=Cn​=0
(16)

のみが成り立つ場合をいう．

二階定数係数線形同次微分方程式¶

各導関数が定数係数を持つ場合の 二階定数係数線形同次微分方程式 は次の形式で書かれる微分方程式である．

d2ydx2+adydx+by=0\frac{d^2y}{dx^2} + a\frac{dy}{dx} + by = 0dx2d2y​+adxdy​+by=0
(17)

ただし，a,ba,ba,bは定数である．

解法¶

この微分方程式の解は特性方程式と呼ばれる形から求める．具体的に，一階の線形同次微分方程式の一般解が y(x)=Ce−∫p(x)dxy(x)= C e^{-\int p(x) dx}y(x)=Ce−∫p(x)dx であったことから，二階の微分方程式の解も指数関数に近い形を持つと考える．そこで，以下のように解を

y=eλxy=e^{\lambda x}y=eλx
(18)

と仮定する．では，このように仮定したとき，どのような λ\lambdaλ であれば微分方程式の解になるかを考える．そのために，この仮定の解を与えられた二階定数係数線形同次微分方程式へ代入する

d2dx2eλx+addxeλx+beλx=0λ2eλx+aλeλx+beλx=0(λ2+aλ+b)eλx=0\begin{aligned} \frac{d^2}{dx^2} e^{\lambda x} + a\frac{d}{dx} e^{\lambda x} + b e^{\lambda x} &= 0 \\ \lambda^2 e^{\lambda x} + a\lambda e^{\lambda x} + b e^{\lambda x} &= 0 \\ \left ( \lambda^2 + a \lambda + b \right ) e^{\lambda x} &= 0 \end{aligned}dx2d2​eλx+adxd​eλx+beλxλ2eλx+aλeλx+beλx(λ2+aλ+b)eλx​=0=0=0​
(19)

以上の方程式が得られ，eλx≠0e^{\lambda x} \neq 0eλx=0 よりこの等式が満たされる条件は以下となる．

λ2+aλ+b=0\lambda^2 + a \lambda + b = 0λ2+aλ+b=0
(20)

この二次方程式は 特性方程式 と呼ばれ，この方程式を満たすような λ\lambdaλ を考えれば良い．つまり，二次方程式の解で学んだように，二階定数係数線形同次微分方程式の一般解は上記の二次方程式の判別式，

D=a2−4bD=a^2 - 4bD=a2−4b
(21)

で判別することができる．では，場合分けを行う．

  1. D=a2−4b>0D=a^2 - 4b > 0D=a2−4b>0

このとき，特性方程式は二つの異なる実根 λα,λβ\lambda_{\alpha}, \lambda_{\beta}λα​,λβ​ を持ち，その一般解は以下となる．

y=C1eλαx+C2eλβxy = C_1 e^{\lambda_{\alpha}x} + C_2 e^{\lambda_{\beta}x}y=C1​eλα​x+C2​eλβ​x
(22)

  1. D=a2−4b=0D=a^2 - 4b = 0D=a2−4b=0

このとき，特性方程式は重根 λα=λβ=−a2\lambda_{\alpha} = \lambda_{\beta} = \frac{-a}{2}λα​=λβ​=2−a​を持ち，その一般解は以下となる．

y=(C1x+C2)eλαxy = \left ( C_1 x + C_2\right )e^{\lambda_{\alpha}x}y=(C1​x+C2​)eλα​x
(23)

  1. D=a2−4b<0D=a^2 - 4b < 0D=a2−4b<0

このとき，特性方程式は複素共役な解 λα=p+qi,λβ=p−qi\lambda_{\alpha}=p+qi, \lambda_{\beta}=p-qiλα​=p+qi,λβ​=p−qi を持ち，その一般解は以下となる．

y=epx(C1cos⁡qx+C2sin⁡qx)y=e^{px} \left( C_1 \cos qx + C_2 \sin qx \right)y=epx(C1​cosqx+C2​sinqx)
(24)

このとき，C1,C2C_1,C_2C1​,C2​は積分定数であり，iii は虚数単位，ppp は実数，qqq は0でない実数である．

基本解と特性方程式

前述したように，微分方程式の解空間は線形空間であり，基本解（基底）を求めることでその解空間全体を表現することができる．基本解は無数にあるので一意に定まるものではなく，線形独立な2つの解が与えられれば，それらは基底となる．二階定数係数線形同次微分方程式では，基本解となるよう，つまりは線形独立となるように解を仮定している．

また，1階線形非同次微分方程式 y′+p(x)y=q(x)y' + p(x)y = q(x)y′+p(x)y=q(x) の解法において現れる定数変化法の仮定

y(x)=C(x)e−∫p(x) dxy(x) = C(x) e^{-\int p(x)\,dx}y(x)=C(x)e−∫p(x)dx
(25)

も，まず対応する同次方程式の解 y(x)=e−∫p(x) dxy(x) = e^{-\int p(x)\,dx}y(x)=e−∫p(x)dx を基本解として得た上で，その係数を定数から関数へと変化させることで非同次項に対応しようとする手法である．したがって，定数変化法においても，まず基本解を用意することが前提となっており，その解が零でないことや線形独立性を満たすことが重要となる．

解答

与えられた二階微分方程式の一般解を y=eλxy=e^{\lambda x}y=eλx とおく．特性方程式を考えると，

d2dx2eλx+3ddxeλx+2eλx=0λ2eλx+3λeλx+2eλx=0(λ2+3λ+2)eλx=0\begin{aligned} \frac{d^2}{dx^2} e^{\lambda x} + 3 \frac{d}{dx} e^{\lambda x} + 2 e^{\lambda x} &= 0 \\ \lambda^2 e^{\lambda x} + 3 \lambda e^{\lambda x} + 2 e^{\lambda x} &= 0 \\ \left ( \lambda^2 + 3 \lambda + 2 \right ) e^{\lambda x} &= 0 \end{aligned}dx2d2​eλx+3dxd​eλx+2eλxλ2eλx+3λeλx+2eλx(λ2+3λ+2)eλx​=0=0=0​
(27)

以上の方程式が得られ，eλx≠0e^{\lambda x} \neq 0eλx=0 よりこの等式が満たされる条件は以下となる．

λ2+3λ+2=0\lambda^2 + 3 \lambda + 2 = 0λ2+3λ+2=0
(28)

この方程式を λ\lambdaλ について解くと，

λ2+3λ+2=(λ+1)(λ+2)=0\lambda^2 + 3 \lambda + 2 = \left( \lambda + 1 \right)\left( \lambda + 2 \right) = 0λ2+3λ+2=(λ+1)(λ+2)=0
(29)

より，

λ=−1,−2\lambda = -1, -2λ=−1,−2
(30)

であり，特性方程式は二つの異なる解を持つので，一般解は

y=C1e−x+C2e−2xy = C_1 e^{-x} + C_2 e^{-2x}y=C1​e−x+C2​e−2x
(31)

となる．
