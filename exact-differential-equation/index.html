完全微分方程式

Google Colab で開く

全微分 とは多変数関数の各変数の微小変化を表現したものである．2変数関数 f(x,y)f(x,y)f(x,y) の全微分 dfdfdf は偏微分を用いて

df=∂f∂xdx+∂f∂ydydf = \frac{\partial f}{\partial x} dx + \frac{\partial f}{\partial y} dydf=∂x∂f​dx+∂y∂f​dy
(1)

と書けることを思い出してほしい．ここで，全微分の形式の形の微分方程式，

p(x,y)dx+q(x,y)dy=0p(x,y)dx+q(x,y)dy = 0p(x,y)dx+q(x,y)dy=0
(2)

を 全微分方程式 という．ここで，p(x,y),q(x,y)p(x,y),q(x,y)p(x,y),q(x,y)はx,yx,yx,yの関数である．そして，全微分方程式の中でもp(x,y),q(x,y)p(x,y),q(x,y)p(x,y),q(x,y)が，

∂f∂x=p(x,y),∂f∂y=q(x,y)\frac{\partial f}{\partial x} = p(x,y), \frac{\partial f}{\partial y} = q(x,y)∂x∂f​=p(x,y),∂y∂f​=q(x,y)
(3)

となるとき，つまり，ある関数 f(x,y)f(x,y)f(x,y) のx,yx,yx,yのそれぞれの偏導関数となっているとき，以下の微分方程式

p(x,y)dx+q(x,y)dy=0p(x,y)dx + q(x,y)dy = 0p(x,y)dx+q(x,y)dy=0
(4)

は 完全微分方程式 と呼ばれる．

完全微分方程式の解法¶

完全微分方程式の一般解の導出は関数 f(x,y)f(x,y)f(x,y) を求めることである．このとき，上記の関数 f(x,y)f(x,y)f(x,y) の全微分式の定義より，

df=p(x,y)dx+q(x,y)dy=0df = p(x,y)dx + q(x,y)dy = 0df=p(x,y)dx+q(x,y)dy=0
(5)

であり，未知関数 f(x,y)f(x,y)f(x,y) は微分したら 0 となるので定数であることがわかる．つまり一般解は，

f(x,y)=Cf(x,y)=Cf(x,y)=C
(6)

となる．ただし，CCCは定数とする．

では，微分方程式 p(x,y)dx+q(x,y)dy=0p(x,y)dx + q(x,y)dy = 0p(x,y)dx+q(x,y)dy=0 を与えられ，未知関数 f(x,y)f(x,y)f(x,y) をどのように求めればいいかを以降で説明する．

まず与えられた微分方程式が完全形であることを確認する．完全微分方程式であるための必要十分条件は

∂p(x,y)∂y=∂q(x,y)∂x\frac{\partial p(x,y)}{\partial y} = \frac{\partial q(x,y)}{\partial x}∂y∂p(x,y)​=∂x∂q(x,y)​
(7)

である．続いて，p(x,y)p(x,y)p(x,y)をxxxについて積分すると

f(x,y)=∫p(x,y)dx+ϕ(y)=Cf(x,y) = \int p(x,y) dx + \phi(y) = Cf(x,y)=∫p(x,y)dx+ϕ(y)=C
(8)

未知関数 f(x,y)f(x,y)f(x,y) の一般解を求めることができるが，yyyを含むのでyyyに関する未知の項 ϕ(y)\phi(y)ϕ(y) が生ずる．ただし，この項は上記のf(x,y)f(x,y)f(x,y)をyyyについて偏微分すると

∂f(x,y)∂y=∂∂y{∫p(x,y)dx}+ddyϕ(y)=q(x,y)\frac{\partial f(x,y)}{\partial y} = \frac{\partial}{\partial y} \left\{ \int p(x,y) dx \right \} + \frac{d}{dy}\phi(y) = q(x,y)∂y∂f(x,y)​=∂y∂​{∫p(x,y)dx}+dyd​ϕ(y)=q(x,y)
(9)

となり，ϕ(y)\phi(y)ϕ(y) の微分を元のp(x,y),q(x,y)p(x,y),q(x,y)p(x,y),q(x,y)の項から関数f(x,y)f(x,y)f(x,y)求めることができる．同一の手順で xxx に関する偏微分を考えても良い．このようにして，偏微分を含む微分方程式であるが解くことができる．

完全微分方程式の具体例¶

次の微分方程式を解いてみよう．

(2x+3y)dx+(3x+8y)dy=0\left ( 2x + 3y \right )dx + \left ( 3x + 8y \right )dy = 0(2x+3y)dx+(3x+8y)dy=0
(10)

解答はクリックで確認できる．

解答

まずは与えられた微分方程式が完全微分方程式であるかを確かめる．p(x,y)=2x+3y,q(x,y)=3x+8yp(x,y)=2x+3y, q(x,y)=3x+8yp(x,y)=2x+3y,q(x,y)=3x+8y としてyyyとxxxに関する偏微分をそれぞれ求めると，

∂p(x,y)∂y=3,∂q(x,y)∂x=3\begin{aligned} \frac{\partial p(x,y)}{\partial y} = 3, \quad \frac{\partial q(x,y)}{\partial x} = 3 \end{aligned}∂y∂p(x,y)​=3,∂x∂q(x,y)​=3​
(11)

となり，∂p(x,y)∂y=∂q(x,y)∂x\frac{\partial p(x,y)}{\partial y} = \frac{\partial q(x,y)}{\partial x}∂y∂p(x,y)​=∂x∂q(x,y)​ であるため完全微分方程式である．

p(x,y)p(x,y)p(x,y) を xxx について積分すると，

f(x,y)=∫p(x,y)dx+ϕ(y)=∫(2x+3y)dx+ϕ(y)=x2+3xy+ϕ(y)\begin{aligned} f(x,y) &= \int p(x,y) dx + \phi(y) \\ &= \int \left( 2x + 3y \right) dx + \phi(y) \\ &= x^2 + 3xy + \phi(y) \end{aligned}f(x,y)​=∫p(x,y)dx+ϕ(y)=∫(2x+3y)dx+ϕ(y)=x2+3xy+ϕ(y)​
(12)

となる．続いて，上記の f(x,y)f(x,y)f(x,y) を yyy について偏微分すると，

∂f(x,y)∂y=∂∂y(x2+3xy+ϕ(y))=3x+ddyϕ(y)=q(x,y)\begin{aligned} \frac{\partial f(x,y)}{\partial y} &= \frac{\partial}{\partial y} \left( x^2 + 3xy + \phi(y) \right ) \\ &= 3x + \frac{d}{dy}\phi(y) = q(x,y) \end{aligned}∂y∂f(x,y)​​=∂y∂​(x2+3xy+ϕ(y))=3x+dyd​ϕ(y)=q(x,y)​
(13)

したがって，ddyϕ(y)=8y\frac{d}{dy}\phi(y)=8ydyd​ϕ(y)=8y とわかる．これを yyy について積分すると，

∫ddyϕ(y)=∫8ydyϕ(y)=4y2\begin{aligned} \int \frac{d}{dy}\phi(y) &= \int 8y dy \\ \phi(y) &= 4y^2 \end{aligned}∫dyd​ϕ(y)ϕ(y)​=∫8ydy=4y2​
(14)

したがって，任意定数 CCC を用いて f(x,y)=Cf(x,y)=Cf(x,y)=C を考慮し，以下の一般解が得られる．

f(x,y)=x2+3xy+4y2=C\begin{aligned} f(x,y) = x^2 + 3xy + 4y^2 = C \end{aligned}f(x,y)=x2+3xy+4y2=C​
(15)
