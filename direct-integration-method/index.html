次の一階常微分方程式を，これまでの微積分の知識を使って解くことを考える．関数 f(x,y)f(x,y)f(x,y) が変数 xxx にのみ依存する関数 g(x)g(x)g(x) である場合，微分方程式は

dydx=g(x)\begin{align} \frac{dy}{dx} = g(x) \end{align}dxdy​=g(x)​​
(1)

と書ける．この形の微分方程式を 直接積分形 と呼ぶ．前述の例は g(x)=xg(x)=xg(x)=x のときの直接積分形である．

直接積分形の解法¶

与えられた一階常微分方程式が上記のような形状に変形可能であるとき，次のステップを踏むことで未知関数 y=y(x)y=y(x)y=y(x) を計算できる．

まず，微分方程式の両辺にdxdxdxをかけると以下のようになる．

dy=g(x)dx\begin{align} dy = g(x)dx \end{align}dy=g(x)dx​​
(2)

そして，式を見てわかるように，xxxとyyyの項がそれぞれ両辺に移項できたことがわかる．続いて両辺を積分する．

∫dy=∫g(x)dx\begin{align} \int dy = \int g(x)dx \end{align}∫dy=∫g(x)dx​​
(3)

この積分を求めると，微分方程式の解である未知関数 y(x)y(x)y(x) が得られる．

y(x)=∫g(x)dx+C\begin{align} y(x) = \int g(x) dx + C \end{align}y(x)=∫g(x)dx+C​​
(4)

ただし，CCCは積分定数である．これは後述する変数分離法で yyy に関する項がないときの特別な場合である．

注意

「dxdxdx を両辺にかける」と説明したが，この操作はdxdxdxとdydydyを微小変化量として仮定したあくまで形式的な記法であり，厳密には両辺の不定積分を行うことを意味している．具体的に，

dydx=g(x)\begin{align} \frac{dy}{dx} = g(x) \end{align}dxdy​=g(x)​​
(5)

という微分方程式より，両辺を積分すると，

∫(1dydx)dx=∫g(x)dx∫dy=∫g(x)dxy=∫g(x)dx+C\begin{align} \int \left(1 \frac{dy}{dx}\right) dx &= \int g(x) dx \\ \int dy &= \int g(x) dx \\ y &= \int g(x) dx + C \end{align}∫(1dxdy​)dx∫dyy​=∫g(x)dx=∫g(x)dx=∫g(x)dx+C​​
(6)

となる．ここで，下記の定理を用いている．このように，微分記号を分数のように扱ってよいのは，積分記号の中で整合的に意味づけされるからである．

定理

y=y(x)y = y(x)y=y(x) は xxx に関する微分可能な関数であり，h(y)h(y)h(y) は yyy に関する連続関数であるとする．このとき，

原始関数

関数 G(x)G(x)G(x) を微分すると g(x)g(x)g(x) になるとき，G(x)G(x)G(x) を g(x)g(x)g(x) の 原始関数 という．すなわち，

直接積分形の具体例¶

解答

まず，両辺に dxdxdx をかけると以下のようになる．

dy=(x2+1)dx\begin{align} dy = \left(x^2 + 1\right) dx \end{align}dy=(x2+1)dx​​

続いて，両辺を積分することを考える．

∫1dy=∫(x2+1)dx\begin{align} \int 1 dy = \int \left(x^2 + 1\right) dx \end{align}∫1dy=∫(x2+1)dx​​

積分すると以下の微分方程式の解が得られる．

y=x33+x+C\begin{align} y = \frac{x^3}{3} + x + C \end{align}y=3x3​+x+C​​

ただし，CCCは積分定数である．実際に結果を微分した結果も確認されたい．与えられた微分方程式になるはずである．

Pythonによる実装¶

上記の微分方程式をSymPyで実装する．前述した式変形はあくまでも手計算での方法であり，SymPyを利用すると（シンボリックに計算できる微分方程式の形式であれば）式変形なしに解を求めることができる．

from sympy import symbols, Eq, Derivative, Function, dsolve

# 変数と未知関数の定義
x = symbols('x')
y = Function('y')(x)

# dy/dxの定義
dy = Derivative(y, x)

# 微分方程式の定義
eq = Eq(dy, x**2+1)

# 微分方程式を解く
general_solution = dsolve(eq, y)
