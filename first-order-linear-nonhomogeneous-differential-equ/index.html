一階線形非同次微分方程式

Google Colab で開く

p(x)p(x)p(x)とq(x)q(x)q(x)は恒等的に0ではない関数として扱うときの一階線形微分方程式を扱う．このとき，

dydx+p(x)y=q(x)\begin{align} \frac{dy}{dx}+p(x)y=q(x) \end{align}dxdy​+p(x)y=q(x)​​
(1)

の形式の微分方程式は一階線形非同次微分方程式と呼ばれる．

特殊解が既知のときの解法¶

一階線形非同次微分方程式の解 yyy を求める際に，その特殊解 zzz が与えられているならば，同次微分方程式に帰着させることができる．では，与えられた一階線形非同次微分方程式が

dydx+p(x)y=q(x)\begin{align} \frac{dy}{dx}+p(x)y=q(x) \end{align}dxdy​+p(x)y=q(x)​​
(2)

として与えられ，その特殊解が zzz ならば，次の方程式も得られる．

dzdx+p(x)z=q(x)\begin{align} \frac{dz}{dx}+p(x)z=q(x) \end{align}dxdz​+p(x)z=q(x)​​
(3)

これらの等式を引き算すると

(dydx+p(x)y)−(dzdx+p(x)z)=q(x)−q(x)(dydx−dzdx)+(y−z)p(x)=0\begin{align} \left(\frac{dy}{dx}+p(x)y\right) - \left(\frac{dz}{dx}+p(x)z\right) &=q(x) - q(x) \\ \left(\frac{dy}{dx} - \frac{dz}{dx}\right) + \left(y - z\right)p(x) &= 0 \end{align}(dxdy​+p(x)y)−(dxdz​+p(x)z)(dxdy​−dxdz​)+(y−z)p(x)​=q(x)−q(x)=0​​
(4)

ここで Y=y−zY = y - zY=y−z として考えると，dYdx=dydx−dzdx\frac{dY}{dx} = \frac{dy}{dx} - \frac{dz}{dx}dxdY​=dxdy​−dxdz​ より

dYdx+p(x)Y=0\begin{align} \frac{dY}{dx} + p(x) Y &= 0 \end{align}dxdY​+p(x)Y​=0​​
(5)

となり，これは一階線形同次微分方程式の形式となる．その一般解は任意定数CCCを用いて

Y=Ce−∫p(x)dx\begin{align} Y &= C e^{-\int p(x) dx} \end{align}Y​=Ce−∫p(x)dx​​
(6)

として表され，Y=y−zY = y - zY=y−zより，本来求めたい関数 yyy について整理すると，

y−z=Ce−∫p(x)dxy=Ce−∫p(x)dx+z\begin{align} y - z &= C e^{-\int p(x) dx} \\ y &= C e^{-\int p(x) dx} + z \end{align}y−zy​=Ce−∫p(x)dx=Ce−∫p(x)dx+z​​
(7)

となり，一階線形非同次微分方程式の解が求めることができた．またこの解を見ると，その解が同次微分方程式の解 Ce−∫p(x)dxC e^{-\int p(x) dx}Ce−∫p(x)dx と特殊解 zzz の和となっていることがわかる．

解の重ね合わせ

非同次微分方程式に加えて，その特殊解が与えられている場合，一階線形微分方程式における 重ね合わせの性質 により，同次微分方程式の一般解と非同次微分方程式の特殊解の和として，非同次微分方程式の一般解を表すことができる．

一般に，以下の一階線形微分方程式

dydx+p(x)y=q(x)\frac{dy}{dx} + p(x)y = q(x)dxdy​+p(x)y=q(x)
(8)

が与えられたとき，対応する同次方程式（すなわち q(x)=0q(x) = 0q(x)=0 のとき）

dydx+p(x)y=0\frac{dy}{dx} + p(x)y = 0dxdy​+p(x)y=0
(9)

の一般解を u(x)u(x)u(x)，非同次方程式（q(x)≠0q(x) \ne 0q(x)=0）の特殊解を v(x)v(x)v(x) とすれば，非同次方程式の一般解 y(x)y(x)y(x) は次のように表される．

y(x)=u(x)+v(x)y(x) = u(x) + v(x)y(x)=u(x)+v(x)
(10)

この構造は，線形微分方程式の解空間が同次解空間の一次平行移動であるからである．

では，次の一階線形非同次微分方程式を解いてみよう．ただし，特殊解として y=x−1y = x - 1y=x−1 が与えられている．

dydx+y=x\begin{align} \frac{dy}{dx}+y=x \end{align}dxdy​+y=x​​
(11)

解答

このとき，y=x−1y = x - 1y=x−1 を考えると，dydx=1\frac{dy}{dx} = 1dxdy​=1 より，上記の微分方程式を満たす解であることがわかる．これを特殊解として考える．非同次微分方程式の一般解は同次微分方程式の一般解と特殊解との和であったので，以下の同次形

dydx+y=0\begin{align} \frac{dy}{dx}+y=0 \end{align}dxdy​+y=0​​
(12)

の一般解が，

y=Ce−x\begin{align} y = Ce^{-x} \end{align}y=Ce−x​​
(13)

であったので特殊解との和をとると，前述の非同次形の一般解は

y=Ce−x+x−1\begin{align} y = Ce^{-x} + x - 1 \end{align}y=Ce−x+x−1​​
(14)

となる．

積分因子による解法¶

この形式の微分方程式は積分因子法で解くことが一般的である．具体的には両辺に積分因子 e∫p(x)dxe^{\int p(x) dx}e∫p(x)dx を乗算すると以下の形に変形できる．

(15)

このとき，左辺の項に着目すると，関数 yyy と e∫p(x)dxe^{\int p(x) dx}e∫p(x)dx の関数の積 ye∫p(x)dxy e^{\int p(x) dx}ye∫p(x)dx の xxx に関する微分であることがわかる（参考：積の微分法則）．

ddxye∫p(x)dx=dydxe∫p(x)dx+yddxe∫p(x)dx=dydxe∫p(x)dx+yp(x)e∫p(x)dx\begin{align} \frac{d}{dx} y e^{\int p(x) dx} &= \frac{dy}{dx} e^{\int p(x) dx} + y \frac{d}{dx} e^{\int p(x) dx} \\ &= \frac{dy}{dx} e^{\int p(x) dx} + y p(x) e^{\int p(x) dx} \\ \end{align}dxd​ye∫p(x)dx​=dxdy​e∫p(x)dx+ydxd​e∫p(x)dx=dxdy​e∫p(x)dx+yp(x)e∫p(x)dx​​
(16)

これを元に前述の式を書き換えると，

dydxe∫p(x)dx+p(x)ye∫p(x)dx=q(x)e∫p(x)dxddxye∫p(x)dx=q(x)e∫p(x)dx\begin{align} \frac{dy}{dx} e^{\int p(x) dx} + p(x)y e^{\int p(x) dx} &= q(x)e^{\int p(x) dx} \\ \frac{d}{dx} y e^{\int p(x) dx} &= q(x)e^{\int p(x) dx} \end{align}dxdy​e∫p(x)dx+p(x)ye∫p(x)dxdxd​ye∫p(x)dx​=q(x)e∫p(x)dx=q(x)e∫p(x)dx​​
(17)

この式をxxxについて積分すると一階線形非同次微分方程式の一般解が得られる．左辺については ddxye∫p(x)dx\frac{d}{dx} y e^{\int p(x) dx}dxd​ye∫p(x)dx が ye∫p(x)dxy e^{\int p(x) dx}ye∫p(x)dx の xxx に関する導関数であるので，それを xxx で積分する，つまり，結果は ye∫p(x)dxy e^{\int p(x) dx}ye∫p(x)dx となることに注意する．

∫ddxye∫p(x)dxdx=∫q(x)e∫p(x)dxdxye∫p(x)dx+C′=∫q(x)e∫p(x)dxdxy=e−∫p(x)dx(∫q(x)e∫p(x)dxdx+C)\begin{align} \int \frac{d}{dx} y e^{\int p(x) dx} dx &= \int q(x)e^{\int p(x) dx} dx \\ y e^{\int p(x) dx} + C' &= \int q(x)e^{\int p(x) dx} dx\\ y &= e^{-\int p(x) dx} \left ( \int q(x)e^{\int p(x) dx} dx + C \right) \end{align}∫dxd​ye∫p(x)dxdxye∫p(x)dx+C′y​=∫q(x)e∫p(x)dxdx=∫q(x)e∫p(x)dxdx=e−∫p(x)dx(∫q(x)e∫p(x)dxdx+C)​​

定数変化法による解法¶

一階線形非同次微分方程式の解を求めるにあたり，同次微分方程式の解を定数変化法というテクニックで変形し，解が同次微分方程式の解と特殊解の和となる性質から一般解を導出方法がある．これを紹介する．

ではまず同次微分方程式，

dydx+p(x)y=0\begin{align} \frac{dy}{dx}+p(x)y=0 \end{align}dxdy​+p(x)y=0​​
(24)

の一般解が

y1=Ce−∫p(x)dx\begin{align} y_1 = C e^{-\int p(x) dx} \end{align}y1​=Ce−∫p(x)dx​​
(25)

で与えられるところからスタートする．前述から解が同次微分方程式の解と特殊解の和となる性質がわかっているので，特殊解を求めるために，ここで積分定数 CCC を変数 xxx の関数 C(x)C(x)C(x) として特殊解 y2y_2y2​ を仮定する．

y2=C(x)e−∫p(x)dx\begin{align} y_2 = C(x) e^{-\int p(x) dx} \end{align}y2​=C(x)e−∫p(x)dx​​
(26)

この同次微分方程式の一般解を求めたい非同次微分方程式に代入し，式変形をする．

として，関数C(x)C(x)C(x)の微分で整理することができた．右辺は変数 xxx の項のみなのでxxxで積分すると，

∫C′(x)dx=∫q(x)e∫p(x)dxdxC(x)=∫q(x)e∫p(x)dxdx\begin{align} \int C'(x) dx &= \int q(x) e^{\int p(x) dx} dx \\ C(x) &= \int q(x) e^{\int p(x) dx} dx \end{align}∫C′(x)dxC(x)​=∫q(x)e∫p(x)dxdx=∫q(x)e∫p(x)dxdx​​

ただし，ここで C′(x)C'(x)C′(x) の積分時に積分定数はこのあとまとめられるので省略する．得られた C(x)C(x)C(x) を y2=C(x)e−∫p(x)dxy_2 = C(x) e^{-\int p(x) dx}y2​=C(x)e−∫p(x)dx に代入すると以下の特殊解 y2y_2y2​ が得られる．

# 変数と未知関数の定義

# 微分方程式の定義

# 微分方程式を解く
y_ = dsolve(eq, y)
