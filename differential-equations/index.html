微分方程式とは？¶

微分方程式とは，独立変数 xxx，未知関数 y(x)y(x)y(x)，そしてその導関数を含む方程式である．

これまでに扱ってきた方程式では，例えば二次方程式

x2−3x+2=0x^2-3x+2=0x2−3x+2=0
(1)

を解くと，x=1,2x=1,2x=1,2 のように数が解として得られる．一方で，微分方程式では未知数ではなく未知関数を求める．例えば，以下のように，

dydx=f(x,y)\frac{dy}{dx} = f(x,y)dxdy​=f(x,y)
(2)

という等式が与えられたとする．この等式は，左辺の y(x)y(x)y(x) の微分 dydx\frac{dy}{dx}dxdy​ が，右辺の xxx と yyy の関数 f(x,y)f(x,y)f(x,y) に等しいことを意味する．したがって，この等式を満たす関数 y(x)y(x)y(x) を求めることが，微分方程式を解くということである．

微分方程式の例¶

具体例から微分方程式を理解しよう．関数 f(x,y)f(x,y)f(x,y) が f(x,y)=xf(x,y)=xf(x,y)=x であるときを考える．このとき得られる微分方程式は一階の常微分方程式として分類される．

dydx=x\frac{dy}{dx} = xdxdy​=x
(3)

この微分方程式は，関数 y(x)y(x)y(x) の導関数が関数 xxx と等しいことを意味する．初めは，慣れないが，微分したら xxx となるような関数を求めることがゴールであり，このような関数は容易に暗算で計算できる．

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(4)

このとき，CCCは積分定数であり，上記の関数 y(x)y(x)y(x) を微分 dydx\frac{dy}{dx}dxdy​ すると xxxとなることがわかるかと思う．この解は任意の定数 CCC に対する解の集合を表しており，これを一般解と呼ぶ．

では上記の微分方程式をSymPyで解いてみる．まずは必要なライブラリをimportする．

from sympy import symbols, Eq, Derivative, Function, dsolve

変数 x と関数 y(x) を定義する．

x = symbols('x')
y = Function('y')(x)

一般解と特解¶

改めて前述の話をまとめる．微分方程式は解として未知関数を持つ方程式であり，上記のように，積分定数を含む形で解の集合を表す形式を一般解と言った．任意の定数 CCC は微分すると消えるので，導出した解には，例えば，y(x)=x2/2y(x) = x^2/2y(x)=x2/2, y(x)=x2/2+1y(x) = x^2/2 + 1y(x)=x2/2+1 などさまざまな関数が含まれる．つまり，一般解はパラメータ CCC を持つ関数の集合なのである．

では，これをSymPyを使ってプロットしてみよう．

from sympy import plotting

C1 = symbols('C1')
solutions = [general_solution.rhs.subs(C1, c) for c in range(-3, 4)]
p = plotting.plot(*solutions, (x, -2, 2), show=False)
p.show()

上記のように，積分定数 C1 に値を入力してみると，複数の解の候補がプロットできる．一般解は，このような解の集合（=関数の集合）を表している．

そして，第一回の講義で紹介したように，問題によっては解に条件が課される場合がある．その条件を 初期条件 と呼び，初期条件に基づいて上記の集合からある特定の解（関数）を求めることができる．このときの解を 特解 という．

前述の例に初期条件を付加すると，

dydx=x,y(0)=1\frac{dy}{dx} = x, \quad y(0) = 1dxdy​=x,y(0)=1
(5)

となる．これはつまり，x=0x=0x=0 のときの未知関数 yyy の値は 1 であるという初期条件である．一般解が以下であったので，

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(6)

この一般解に y(0)=1y(0) = 1y(0)=1 を代入する．

y(0)=022+C1=0+CC=1\begin{align} y(0) &= \frac{0^2}{2} + C \\ 1 &= 0 + C \\ C &= 1 \end{align}y(0)1C​=202​+C=0+C=1​​
(7)

となり，積分定数 CCC は 1 であり，特解は

y(x)=x22+1y(x) = \frac{x^2}{2} + 1y(x)=2x2​+1
(8)

である．

SymPyのdsolveに初期条件を与えて特解を計算するためには，icsというキーワード引数で辞書形式で初期条件を与えれば良い．

particular_solution = dsolve(eq, y, ics={y.subs(x,0): 1})

これをプロットすると以下となる．

境界条件

微分方程式をある区間 [xα,xβ][x_{\alpha}, x_{\beta}][xα​,xβ​] で考えるとき，区間の両端 x=xαx=x_\alphax=xα​，x=xβx=x_\betax=xβ​ における解がみたす条件，つまり，y(xα)y(x_\alpha)y(xα​) と y(xβ)y(x_\beta)y(xβ​) を 境界条件 とよぶ．そして，この条件をみたす解を求める問題を 境界値問題 という．例えば，二階微分方程式 y′′=f(x,y,y′)y''=f(x,y,y')y′′=f(x,y,y′) に対して y(xα)=Ay(x_\alpha)=Ay(xα​)=A，y(xβ)=By(x_\beta)=By(xβ​)=B を与える問題が境界値問題である．

微分方程式の解の確認¶

得られた解がその微分方程式の解であることを確認するには，これまでの方程式と同様に，等式を満たすことを確認すればよい．

ここまでで，微分方程式

dydx=x\frac{dy}{dx} = xdxdy​=x
(9)

の一般解が

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(10)

であることが計算できた．実際にこの関数が微分方程式を満たすか確認してみよう．

得られた一般解を微分する．

ddxy(x)=x\frac{d}{dx}y(x) = xdxd​y(x)=x
(11)

これは与えられた微分方程式に代入しても明らかであるように等式を満たす．よって微分方程式の解である．

解の集合

この確認からも明らかなように，y(x)=x2/2y(x) = x^2/2y(x)=x2/2, y(x)=x2/2+1y(x) = x^2/2 + 1y(x)=x2/2+1 などさまざまな関数を解として方程式を満たすことがわかる．これを解の集合という．なお，同次線形微分方程式の解の集合は線形空間になるため，解空間と呼ぶことがある．

具体例¶

y=e−xy=e^{-x}y=e−xは以下の微分方程式の解であることを確認する．

dydx+y=0\begin{align} \frac{dy}{dx} + y = 0 \end{align}dxdy​+y=0​​
(12)

まず解として与えられた yyy を微分する．

dydx=−e−x\begin{align} \frac{dy}{dx} = - e^{-x} \end{align}dxdy​=−e−x​​
(13)

これと y=e−xy=e^{-x}y=e−x を与えられた微分方程式に代入する．

−e−x+e−x=0- e^{-x} + e^{-x} = 0−e−x+e−x=0
(14)

この式からも明らかなように等式を満たす．従って，y=e−xy=e^{-x}y=e−xは与えられた微分方程式の解である．

微分方程式の形¶

ここまで見た例は一階の常微分方程式であった．微分方程式は大まかに，

  1. 関数と導関数が線形性を持つか？

  2. 独立変数の数は2つ以上か？ に応じて分類することができる．

線形微分方程式¶

まず一つ目の「関数と導関数が線形性を持つか？」について考える．一つの独立変数 xxx を持つ微分方程式は一般に

F(x,y,y′,…,y(n))=0F\left(x,y,y',\ldots,y^{(n)}\right)=0F(x,y,y′,…,y(n))=0
(15)

と表せる．このうち，未知関数 yyy とその導関数に対して，微分方程式が

an(x)y(n)+an−1(x)y(n−1)+⋯+a1(x)y′+a0(x)y=g(x)a_n(x)y^{(n)}+a_{n-1}(x)y^{(n-1)}+\cdots+a_1(x)y'+a_0(x)y=g(x)an​(x)y(n)+an−1​(x)y(n−1)+⋯+a1​(x)y′+a0​(x)y=g(x)
(16)

と書けるとする．ただし，a0(x),a1(x),…,an(x)a_0(x),a_1(x),\ldots,a_n(x)a0​(x),a1​(x),…,an​(x) と g(x)g(x)g(x) は xxx の関数であり，係数には未知関数 yyy やその導関数は含まれない．この左辺を L[y]L[y]L[y] とおくと，

L[y]=an(x)y(n)+an−1(x)y(n−1)+⋯+a1(x)y′+a0(x)yL[y]=a_n(x)y^{(n)}+a_{n-1}(x)y^{(n-1)}+\cdots+a_1(x)y'+a_0(x)yL[y]=an​(x)y(n)+an−1​(x)y(n−1)+⋯+a1​(x)y′+a0​(x)y
(17)

と表せる．このとき，任意の関数 y1,y2y_1,y_2y1​,y2​ と実定数 α,β\alpha,\betaα,β に対して，

L[αy1+βy2]=αL[y1]+βL[y2]\begin{align} L[\alpha y_1+\beta y_2]=\alpha L[y_1]+\beta L[y_2] \end{align}L[αy1​+βy2​]=αL[y1​]+βL[y2​]​​
(18)

が成立すれば，LLL は yyy について線形であり，このような微分方程式を線形微分方程式と呼ぶ．特に g(x)=0g(x)=0g(x)=0 のときは同次線形微分方程式，g(x)≠0g(x)\neq0g(x)=0 のときは非同次線形微分方程式である．この性質が成立しない微分方程式は非線形微分方程式と呼ばれる．非線形微分方程式は簡単にいうとy2y^2y2やyy′yy'yy′といった非線形項を含む微分方程式と言える．

微分作用素

L[y]L[y]L[y] は，関数 y(x)y(x)y(x) に対して微分や足し算を行う操作を表す記号である．例えば L[y]=y′−yL[y]=y'-yL[y]=y′−y と定義すると，LLL は関数 y(x)y(x)y(x) を受け取り，y′(x)−y(x)y'(x)-y(x)y′(x)−y(x) を返す．このように関数を入力として受け取る操作を 作用素 と呼び，特に微分を含むものを 微分作用素 と呼ぶ．

汎関数

関連する概念として，関数を入力として受け取り，数を返すものを 汎関数 と呼ぶ．例えば J[y]=∫01y(x)2 dxJ[y]=\int_0^1 y(x)^2\,dxJ[y]=∫01​y(x)2dx は関数 y(x)y(x)y(x) を受け取り，積分によって一つの数を返すので汎関数である．つまり，作用素は関数から関数，汎関数は関数から数で入力が関数となるが出力が異なることに注意されたい．

線形微分方程式の具体例¶

以下の微分方程式が線形であることを確かめる．

y′−y=xy' - y = xy′−y=x
(19)

解答

前述の線形性の定義を満たすことを確認すればよいので，左辺の微分作用素を次のように定義する．

L[y]=y′−yL[y]=y'-yL[y]=y′−y
(20)

このとき，任意の関数 y1,y2y_1,y_2y1​,y2​ と実定数 α,β\alpha,\betaα,β に対して，

L[αy1+βy2]=(αy1+βy2)′−(αy1+βy2)=αy1′+βy2′−αy1−βy2=α(y1′−y1)+β(y2′−y2)=αL[y1]+βL[y2]\begin{align} L[\alpha y_1+\beta y_2] &= (\alpha y_1+\beta y_2)'-(\alpha y_1+\beta y_2) \\ &= \alpha y_1' + \beta y_2' - \alpha y_1 - \beta y_2 \\ &= \alpha (y_1'-y_1)+\beta (y_2'-y_2) \\ &= \alpha L[y_1]+\beta L[y_2] \end{align}L[αy1​+βy2​]​=(αy1​+βy2​)′−(αy1​+βy2​)=αy1′​+βy2′​−αy1​−βy2​=α(y1′​−y1​)+β(y2′​−y2​)=αL[y1​]+βL[y2​]​​
(21)

となる．したがって，LLL は yyy について線形であり，y′−y=xy'-y=xy′−y=x は非同次線形微分方程式である．

非線形微分方程式の具体例¶

以下の微分方程式が非線形であることを確かめる．

y′′=−y−y2y'' = - y - y^2y′′=−y−y2
(22)

解答はクリックで確認できる．

解答

前述の線形性の定義を満たすことを確認すればよいので微分方程式を次のように定義する．

F(x,y,y′′)=y′′+y+y2F(x,y,y'') = y'' + y + y^2F(x,y,y′′)=y′′+y+y2
(23)

上式の左辺について，

F(x,αy1+βy2,αy1′′+βy2′′)=αy1′′+βy2′′+αy1+βy2+(αy1+βy2)2=αy1′′+βy2′′+αy1+βy2+α2y12+2αβy1y2+β2y22\begin{align} F\left(x, \alpha y_1+\beta y_2, \alpha y_1''+\beta y_2''\right) &= \alpha y_1'' + \beta y_2'' + \alpha y_1+\beta y_2 + (\alpha y_1+\beta y_2)^2 \\ &= \alpha y_1'' + \beta y_2'' + \alpha y_1+\beta y_2 + \alpha^2 y_1^2 + 2\alpha\beta y_1y_2 + \beta^2 y_2^2 \end{align}F(x,αy1​+βy2​,αy1′′​+βy2′′​)​=αy1′′​+βy2′′​+αy1​+βy2​+(αy1​+βy2​)2=αy1′′​+βy2′′​+αy1​+βy2​+α2y12​+2αβy1​y2​+β2y22​​​
(24)

となる．続いて，右辺について確認する．

αF(x,y1,y1′′)+βF(x,y2,y2′′)=α(y1′′+y1+y12)+β(y2′′+y2+y22)=αy1′′+βy2′′+αy1+βy2+αy12+βy22\begin{align} \alpha F \left(x, y_1, y_1''\right)+\beta F\left(x, y_2, y_2''\right) &= \alpha \left( y_1'' + y_1 + y_1^2 \right) + \beta \left(y_2'' + y_2 + y_2^2 \right) \\ &= \alpha y_1'' + \beta y_2'' + \alpha y_1 + \beta y_2 + \alpha y_1^2 + \beta y_2^2 \end{align}αF(x,y1​,y1′′​)+βF(x,y2​,y2′′​)​=α(y1′′​+y1​+y12​)+β(y2′′​+y2​+y22​)=αy1′′​+βy2′′​+αy1​+βy2​+αy12​+βy22​​​
(25)

以上より，両辺の係数が一致しない．従って，yyy について非線形であり，これは非線形微分方程式である．

常微分方程式¶

続いて，二つ目の「独立変数の数は2つ以上か？」について，一つの独立変数を持つ微分方程式を 常微分方程式（Ordinary Differential Equations, ODEs） と呼び，以下の形で与えられる．

F(x,y,y′,…,y(n))=F(x,y,dydx,…,dnydxn)=0\begin{align} F\left(x, y, y',\ldots, y^{(n)}\right) =F\left(x, y, \frac{dy}{dx},\ldots,\frac{d^n y}{dx^n}\right)=0 \end{align}F(x,y,y′,…,y(n))=F(x,y,dxdy​,…,dxndny​)=0​​
(26)

このとき，復習であるがy(n)y^{(n)}y(n)はnnn階の導関数であり，微分方程式に含まれる導関数の最高階 nnn を微分方程式の階数と呼ぶ．上記の式には一つの独立変数 xxx と関数y(x)y(x)y(x)，またその導関数 {y′,…,y(n)}\{y',\ldots,y^{(n)}\}{y′,…,y(n)}が含まれている．

偏微分方程式¶

常微分方程式とは異なり，二つ以上の独立変数を持つ微分方程式を 偏微分方程式（Partial Differential Equations, PDEs） と呼ぶ．偏微分方程式は関数の偏微分に関する等式であり，以下で与えられる．

F(x1,x2,…,xn,u,∂u∂x1,…,∂u∂xn,∂2u∂x12,∂2u∂x1∂x2,…)=0\begin{align} F\left(x_1, x_2, \ldots, x_n, u, \frac{\partial u}{\partial x_1}, \ldots, \frac{\partial u}{\partial x_n}, \frac{\partial^2 u}{\partial x_1^2}, \frac{\partial^2 u}{\partial x_1\partial x_2}, \ldots\right) = 0 \end{align}F(x1​,x2​,…,xn​,u,∂x1​∂u​,…,∂xn​∂u​,∂x12​∂2u​,∂x1​∂x2​∂2u​,…)=0​​
(27)

ここで，FFFはある関数，x1,x2,…,xnx_1, x_2, \ldots, x_nx1​,x2​,…,xn​はnnn個の独立変数，関数uuuは求める未知関数を表す．また，∂u∂xi\frac{\partial u}{\partial x_i}∂xi​∂u​はuuuのxix_ixi​に対する一階の偏微分を表し，二階以上の偏微分や混合偏微分を含むこともある．常微分方程式とは異なり，複数個の独立変数とその偏導関数が含まれていることを確認されたい．

補足：カラーマップ¶

SymPyでの解のプロットにあたり，各解のカラーはSymPyのplotting（=Matplotlib）のデフォルト値になっている．プロットする際に次のようにカラーマップを取得して，色を指定すると見栄えの良いグラフを描くことができる．

import numpy as np
from sympy import plotting
import matplotlib.pyplot as plt

general_solution = dsolve(eq, y) # 一般解の計算
C1 = symbols('C1')

N = 7 # -3 から 3 までの計7本のグラフを描くのでN=7

eqs = []
for c in np.linspace(-3, 3, N):
    eqs.append(general_solution.rhs.subs(C1, c))
p = plotting.plot(*eqs, (x, -2, 2), show=False)

cm = plt.get_cmap('magma', N) # magmaカラーマップを使用
for i in range(cm.N):
    p[i].line_color = cm(i)
p.show()
