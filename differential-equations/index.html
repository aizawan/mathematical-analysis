微分方程式とは？¶

微分方程式とは，変数 xxx と変数 xxx を持つ未知関数 yyy とその導関数を含む方程式である．例えば，以下のように，

dydx=f(x,y)\frac{dy}{dx} = f(x,y)dxdy​=f(x,y)
(1)

という等式が与えられ，この等式は左辺の独立変数 xxx を持つ関数 y(x)y(x)y(x) の微分 dydx\frac{dy}{dx}dxdy​ が右辺の変数 x,yx,yx,y を持つ関数 f(x,y)f(x,y)f(x,y) を持つ関数が等しいことを意味し，これまで学んだ x=1x=1x=1 といった解が得られる方程式とは異なり，この等式を満たす関数 y(x)y(x)y(x) を求めることが微分方程式を解くということである．

微分方程式の形¶

微分方程式は大まかに，

  1. 関数と導関数が線形性を持つか？

  2. 独立変数の数は2つ以上か？ に応じて分類することができる．

線形微分方程式¶

まず一つ目の「関数と導関数が線形性を持つか？」について，微分方程式を

F(x,y,y′)=0F(x,y,y')=0F(x,y,y′)=0
(2)

と定義する．ここでy′=dydxy'=\frac{dy}{dx}y′=dxdy​であり，実定数 α,β\alpha,\betaα,βに対して，

F(x,αy1+βy2,αy1′+βy2′)=αF(x,y1,y1′)+βF(x,y2,y2′)\begin{align} F\left(x, \alpha y_1+\beta y_2, \alpha y_1'+\beta y_2'\right)= \alpha F \left(x, y_1, y_1'\right)+\beta F\left(x, y_2, y_2'\right) \end{align}F(x,αy1​+βy2​,αy1′​+βy2′​)=αF(x,y1​,y1′​)+βF(x,y2​,y2′​)​​
(3)

が成立すれば微分方程式が線形であり，その微分方程式を線形微分方程式と呼ぶ．成立しない微分方程式は非線形微分方程式と呼ばれる．非線形微分方程式は簡単にいうとy2y^2y2やyy′yy'yy′といった非線形項を含む微分方程式と言える．

線形微分方程式の具体例¶

以下の微分方程式が線形であることを確かめる．

y′−y=xy' - y = xy′−y=x
(4)

解答はクリックで確認できる．

解答

前述の線形性の定義を満たすことを確認すればよいので微分方程式を次のように定義する．

F(x,y,y′)=y′−y−x=0F(x,y,y') = y' - y - x = 0F(x,y,y′)=y′−y−x=0
(5)

上式の左辺について，

F(x,αy1+βy2,αy1′+βy2′)=αy1′+βy2′−(αy1+βy2)−x\begin{align} F\left(x, \alpha y_1+\beta y_2, \alpha y_1'+\beta y_2'\right) &= \alpha y_1' + \beta y_2 ' - (\alpha y_1+\beta y_2) - x \end{align}F(x,αy1​+βy2​,αy1′​+βy2′​)​=αy1′​+βy2′​−(αy1​+βy2​)−x​​
(6)

となる．続いて，右辺について確認する．

αF(x,y1,y1′)+βF(x,y2,y2′)=α(y1′−y1−x)+β(y2′−y2−x)=αy1′+βy2′−αy1−βy2−αx−βx=αy1′+βy2′−(αy1+βy2)−(α+β)x\begin{align} \alpha F \left(x, y_1, y_1'\right)+\beta F\left(x, y_2, y_2'\right) &= \alpha \left(y_1' - y_1 - x \right) + \beta \left(y_2' - y_2 - x \right) \\ &= \alpha y_1' + \beta y_2' - \alpha y_1 - \beta y_2 - \alpha x - \beta x \\ &= \alpha y_1' + \beta y_2' - \left(\alpha y_1 + \beta y_2\right) - \left(\alpha + \beta\right) x \\ \end{align}αF(x,y1​,y1′​)+βF(x,y2​,y2′​)​=α(y1′​−y1​−x)+β(y2′​−y2​−x)=αy1′​+βy2′​−αy1​−βy2​−αx−βx=αy1′​+βy2′​−(αy1​+βy2​)−(α+β)x​​
(7)

以上より，y,y′y,y'y,y′の項について両辺が一致するので yyy について線形であり，これは線形微分方程式である．

非線形微分方程式の具体例¶

以下の微分方程式が非線形であることを確かめる．

y′′=−y−y2y'' = - y - y^2y′′=−y−y2
(8)

解答はクリックで確認できる．

解答

前述の線形性の定義を満たすことを確認すればよいので微分方程式を次のように定義する．

F(x,y,y′′)=y′′+y+y2F(x,y,y'') = y'' + y + y^2F(x,y,y′′)=y′′+y+y2
(9)

上式の左辺について，

F(x,αy1+βy2,αy1′′+βy2′′)=αy1′′+βy2′′+αy1+βy2+(αy1)2+(βy2)2=αy1′′+βy2′′+αy1+βy2+α2y12+β2y22\begin{align} F\left(x, \alpha y_1+\beta y_2, \alpha y_1''+\beta y_2''\right) &= \alpha y_1'' + \beta y_2 '' + \alpha y_1+\beta y_2 + (\alpha y_1)^2 + (\beta y_2)^2 \\ &= \alpha y_1'' + \beta y_2 '' + \alpha y_1+\beta y_2 + \alpha^2 y_1^2 + \beta^2 y_2^2 \end{align}F(x,αy1​+βy2​,αy1′′​+βy2′′​)​=αy1′′​+βy2′′​+αy1​+βy2​+(αy1​)2+(βy2​)2=αy1′′​+βy2′′​+αy1​+βy2​+α2y12​+β2y22​​​
(10)

となる．続いて，右辺について確認する．

αF(x,y1,y1′′)+βF(x,y2,y2′′)=α(y1′′+y1+y12)+β(y2′′+y2+y22)=αy1′′+βy2′′+αy1+βy2+αy12+βy22\begin{align} \alpha F \left(x, y_1, y_1''\right)+\beta F\left(x, y_2, y_2''\right) &= \alpha \left( y_1'' + y_1 + y_1^2 \right) + \beta \left(y_2'' + y_2 + y_2^2 \right) \\ &= \alpha y_1'' + \beta y_2'' + \alpha y_1 + \beta y_2 + \alpha y_1^2 + \beta y_2^2 \end{align}αF(x,y1​,y1′′​)+βF(x,y2​,y2′′​)​=α(y1′′​+y1​+y12​)+β(y2′′​+y2​+y22​)=αy1′′​+βy2′′​+αy1​+βy2​+αy12​+βy22​​​
(11)

以上より，両辺の係数が一致しない．従って，yyy について非線形であり，これは非線形微分方程式である．

常微分方程式¶

続いて，二つ目の「独立変数の数は2つ以上か？」について，一つの独立変数を持つ微分方程式を 常微分方程式（Ordinary Differential Equations, ODEs） と呼び，以下の形で与えられる．

F(x,y,y′,...,y(n))=F(x,y,dydx,...,dnydxn)=0\begin{align} F\left(x, y, y',..., y^{(n)}\right) =F\left(x, y, \frac{dy}{dx},...,\frac{d^ny}{dx^n}\right)=0 \end{align}F(x,y,y′,...,y(n))=F(x,y,dxdy​,...,dxndny​)=0​​
(12)

このとき，復習であるがy(n)y^{(n)}y(n)はnnn階の導関数であり，微分方程式に含まれる導関数の最大の次数 nnn を微分方程式の階数と呼ぶ．上記の式には一つの独立変数 xxx と関数y(x)y(x)y(x)，またその導関数 {y′,...yn}\{y',...y^{n}\}{y′,...yn}が含まれている．

偏微分方程式¶

常微分方程式とは異なり，二つ以上の独立変数を持つ微分方程式を 偏微分方程式（Partial Differential Equations, PDEs） と呼ぶ．偏微分方程式は関数の偏微分に関する等式であり，以下で与えられる．

F(x1,x2,...,xn,u,∂u∂x1,∂u∂x2,...,∂u∂xn,∂2u∂x12,∂2u∂x22,...,∂2u∂xn2,...)=0\begin{align} F\left(x_1, x_2, ..., x_n, u, \frac{\partial u}{\partial x_1}, \frac{\partial u}{\partial x_2}, ..., \frac{\partial u}{\partial x_n}, \frac{\partial^2 u}{\partial x_1^2}, \frac{\partial^2 u}{\partial x_2^2}, ..., \frac{\partial^2 u}{\partial x_n^2}, ...\right) = 0 \end{align}F(x1​,x2​,...,xn​,u,∂x1​∂u​,∂x2​∂u​,...,∂xn​∂u​,∂x12​∂2u​,∂x22​∂2u​,...,∂xn2​∂2u​,...)=0​​
(13)

ここで，FFFはある関数，x1,x2,...,xnx_1, x_2, ..., x_nx1​,x2​,...,xn​はnnn個の独立変数，関数uuuは求める未知関数を表す．また，∂u∂xi\frac{\partial u}{\partial x_i}∂xi​∂u​はuuuのxix_ixi​に対する一階の偏微分を表し，nnn階の偏微分を含む．常微分方程式とは異なり，複数個の独立変数とその偏導関数が含まれていることを確認されたい．

微分方程式の例¶

具体例から微分方程式を理解しよう．関数 f(x,y)f(x,y)f(x,y) が f(x,y)=xf(x,y)=xf(x,y)=x であるときを考える．この関数は一階の常微分方程式として分類される．

dydx=x\frac{dy}{dx} = xdxdy​=x
(14)

この微分方程式は，関数 y(x)y(x)y(x) の導関数が関数 xxx と等しいことを意味する．初めは，慣れないが，微分したら xxx となるような関数を求めることがゴールであり，このような関数は容易に暗算で計算できる．

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(15)

このとき，CCCは積分定数であり，上記の関数 y(x)y(x)y(x) を微分 dydx\frac{dy}{dx}dxdy​ すると xxxとなることがわかるかと思う．この解は任意の定数 CCC に対する解の集合を表しており，これを一般解と呼ぶ．

では上記の微分方程式をsympyで解いてみる．まずは必要なライブラリをimportする．

from sympy import symbols, Eq, Derivative, Function, dsolve

一般解と特殊解¶

改めて前述の話をまとめる．微分方程式は解として未知関数を持つ方程式であり，上記のように，積分定数を含む形で解の集合を表す形式を一般解と言った．ここでの解の集合は任意の定数 CCC は微分して消えるのでどのような値が入っていもいいと考えると，導出した解は，例えば，y(x)=x2/2y(x) = x^2/2y(x)=x2/2, y(x)=x2/2+1y(x) = x^2/2 + 1y(x)=x2/2+1 などさまざまな関数を解として持つことがわかる．つまり，一般解はパラメータ CCC を持つ関数の集合なのである．

では，これをsympyを使ってプロットしてみよう．

from sympy import plotting
plotting.plot(
    y_.rhs.subs(symbols('C1'), -3),
    y_.rhs.subs(symbols('C1'), -2),
    y_.rhs.subs(symbols('C1'), -1),
    y_.rhs.subs(symbols('C1'), 0),
    y_.rhs.subs(symbols('C1'), 1),
    y_.rhs.subs(symbols('C1'), 2),
    y_.rhs.subs(symbols('C1'), 3), (x, -2, 2))
<sympy.plotting.backends.matplotlibbackend.matplotlib.MatplotlibBackend at 0x108f81340>

上記のように，積分定数 C1 に値を入力してみると，複数の解の候補がプロットできる．これが一般解であり，解の集合（=関数の集合）ということである．これを解空間といったりする．

そして，第一回の講義で紹介したように，問題によっては解に条件が課される場合がある．その条件を 初期条件 と呼び，初期条件に基づいて上記の集合からある特定の解（関数）を求めることができる．このときの解を 特殊解 という．

前述の例に初期条件を付加すると，

dydx=x,y(0)=1\frac{dy}{dx} = x, \quad y(0) = 1dxdy​=x,y(0)=1
(16)

となる．これはつまり，x=0x=0x=0 のときの未知関数 yyy の値は 1 であるという初期条件である．一般解が以下であったので，

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(17)

この一般解に y(0)=1y(0) = 1y(0)=1 を代入する．

y(0)=022+C1=0+CC=1\begin{align} y(0) &= \frac{0^2}{2} + C \\ 1 &= 0 + C \\ C &= 1 \end{align}y(0)1C​=202​+C=0+C=1​​
(18)

となり，積分定数 CCC は 1 であり，特殊解は

y(x)=x22+1y(x) = \frac{x^2}{2} + 1y(x)=2x2​+1
(19)

である．

sympyのdsolveに初期条件を与えて特殊解を計算するためには，icsというキーワード引数で辞書形式で初期条件を与えれば良い．

y_ = dsolve(eq, y, ics={y.subs(x,0): 1})

境界条件

微分方程式をある区間 [xα,xβ][x_{\alpha}, x_{\beta}][xα​,xβ​] で考えるとき，区間の両端 x=xαx=x_\alphax=xα​，x=xβx=x_\betax=xβ​ における解がみたす条件，つまり，y(xα)y(x_\alpha)y(xα​) と y(xβ)y(x_\beta)y(xβ​) を 境界条件 とよぶ．そして，この条件をみたす解を求める問題を 境界値問題 という．

微分方程式の解の確認¶

得られた解がその微分方程式の解であることを確認するには，これまでの方程式と同様に，等式を満たすことを確認すればよい．

ここまでで，微分方程式

dydx=x\frac{dy}{dx} = xdxdy​=x
(20)

の一般解が

y(x)=x22+Cy(x) = \frac{x^2}{2} + Cy(x)=2x2​+C
(21)

であることが計算できた．実際にこの関数が微分方程式を満たすか確認してみよう．

得られた一般解を微分する．

ddxy(x)=x\frac{d}{dx}y(x) = xdxd​y(x)=x
(22)

これは与えられた微分方程式に代入しても明らかであるように等式を満たす．よって微分方程式の解である．

解空間

この確認からも明らかなように，y(x)=x2/2y(x) = x^2/2y(x)=x2/2, y(x)=x2/2+1y(x) = x^2/2 + 1y(x)=x2/2+1 などさまざまな関数を解として方程式を満たすことがわかる．これを解の集合（解空間）といった．

具体例¶

y=e−xy=e^{-x}y=e−xは以下の微分方程式の解であることを確認する．

dydx+y=0\begin{align} \frac{dy}{dx} + y = 0 \end{align}dxdy​+y=0​​
(23)

解答はクリックで確認できる．

解答

まず解として与えられた yyy を微分する．

dydx=−e−x\begin{align} \frac{dy}{dx} = - e^{-x} \end{align}dxdy​=−e−x​​
(24)

これと y=e−xy=e^{-x}y=e−x を与えられた微分方程式に代入する．

−e−x+e−x=0- e^{-x} + e^{-x} = 0−e−x+e−x=0
(25)

この式からも明らかなように等式を満たす．従って，y=e−xy=e^{-x}y=e−xは与えられた微分方程式の解である．

補足：カラーマップ¶

Sympyでの解のプロットにあたり，各解のカラーはSympyのplotting（=Matplotlib）のデフォルト値になっている．プロットする際に次のようにカラーマップを取得して，色を指定すると見栄えの良いグラフを描くことができる．

import numpy as np
from sympy import plotting
import matplotlib.pyplot as plt

y_ = dsolve(eq, y) # 一般解の計算

N = 7 # -3 から 3 までの計7本のグラフを書くのでN=7

eqs = []
for c in np.linspace(-3, 3, N):
    eqs.append(y_.rhs.subs(symbols('C1'), c))
p = plotting.plot(*eqs, (x, -2, 2), show=False)

cm = plt.get_cmap('magma', N) # magmaカラーマップを使用
for i in range(cm.N):
    p[i].line_color = cm(i)
p.show()
