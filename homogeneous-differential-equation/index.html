dydx=f(yx)\begin{align} \frac{dy}{dx} = f\left(\frac{y}{x}\right) \end{align}dxdy​=f(xy​)​​
(1)

ただし，x≠0x\neq 0x=0 の区間で u=yxu=\frac{y}{x}u=xy​ とおく．f(u)−u≠0f(u)-u\neq 0f(u)−u=0 の範囲では，一般解は以下で与えられる．

∫1f(u)−u du=log⁡∣x∣+C\begin{align} \int \frac{1}{f(u) -u}\,du= \log |x| + C \end{align}∫f(u)−u1​du=log∣x∣+C​​
(2)

ただし，CCCは積分定数とする．また，f(a)=af(a)=af(a)=a を満たす定数 aaa がある場合は，u=au=au=a，すなわち y=axy=axy=ax も解になる．

同次式

同次式は 斉次式（せいじしき） ともいう．定数 λ\lambdaλ について，0 以上の整数 nnn ，および関数 f(x,y)f(x,y)f(x,y) に対して，

f(λx,λy)=λnf(x,y)f(\lambda x, \lambda y) = \lambda^n f(x, y)f(λx,λy)=λnf(x,y)
(3)

が成立するとき，関数 fff は nnn 次の同次式という．例えば，

f(x,y)=x2+xy+y2f(x,y)=x^2+xy+y^2f(x,y)=x2+xy+y2
(4)

については，

f(λx,λy)=(λx)2+(λx)(λy)+(λy)2=λ2(x2+xy+y2)=λ2f(x,y)\begin{align} f(\lambda x, \lambda y) &=(\lambda x)^2+(\lambda x)(\lambda y)+(\lambda y)^2 \\ &=\lambda^2(x^2+xy+y^2) \\ &=\lambda^2 f(x,y) \end{align}f(λx,λy)​=(λx)2+(λx)(λy)+(λy)2=λ2(x2+xy+y2)=λ2f(x,y)​​
(5)

となるため，2 次の同次式である．また，g(x,y)=3x−2yg(x,y)=3x-2yg(x,y)=3x−2y は 1 次の同次式である．

なお，このページで扱う同次微分方程式は，右辺が yx\frac{y}{x}xy​ の関数として書ける一階微分方程式を指している．したがって，右辺が 0 でなくてもこの意味では同次形と呼ぶ．一方，線形微分方程式でいう同次・非同次は，非同次項が 0 かどうかによる別の分類である．

Pythonによる実装¶

では，これをsympyで解く．

from sympy import symbols, Eq, Derivative, Function, dsolve

# 変数，関数，導関数を定義．
x = symbols('x')
y = Function('y')(x)
dy = Derivative(y, x)

# 与えられた同次微分方程式を定義
eq = Eq(dy, (x**2 + y**2) / (x*y))

# 微分方程式を解く．
y_ = dsolve(eq, y)

このとき手計算した一般解からもわかるように，解は正負の2つの枝を持つことに注意する．また，sympy の結果は log⁡(x)\log(x)log(x) を含むため，ここでは x>0x>0x>0 の区間で表示する．具体的には，積分定数 C1 を 0≤C1≤30 \leq C_1 \leq 30≤C1​≤3，横軸を 1≤x≤21 \leq x \leq 21≤x≤2 としている．この範囲では，平方根の中身 C1+2log⁡(x)C_1+2\log(x)C1​+2log(x) が負にならない．
