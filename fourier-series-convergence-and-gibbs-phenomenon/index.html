フーリエ級数の収束とギブス現象

Google Colab で開く

フーリエ級数展開では，関数 f(x)f(x)f(x) を三角関数の和で表した．しかし，フーリエ級数の式

f(x)∼a02+∑k=1∞(akcos⁡kx+bksin⁡kx)\begin{align} f(x) \sim \frac{a_0}{2}+\sum_{k=1}^{\infty}\left(a_k\cos kx+b_k\sin kx\right) \end{align}f(x)∼2a0​​+k=1∑∞​(ak​coskx+bk​sinkx)​​
(1)

に現れる記号 ∼\sim∼ は，必ずすべての点で右辺が f(x)f(x)f(x) と一致することを意味しているわけではない．特に不連続点をもつ関数では，フーリエ級数がどの値へ収束するのかを確認する必要がある．

このノートでは，フーリエ級数の収束先を与える ディリクレ・ジョルダンの定理 を紹介し，不連続点の近くで現れる ギブス現象 を Python で確認する．

フーリエ級数の部分和¶

無限和を一度に計算することはできないので，実際には有限個の項で打ち切った部分和を考える．区間 [−π,π][-\pi,\pi][−π,π] 上の関数 f(x)f(x)f(x) に対して，NNN 次の部分和を

SN(x)=a02+∑k=1N(akcos⁡kx+bksin⁡kx)\begin{align} S_N(x)=\frac{a_0}{2}+\sum_{k=1}^{N}\left(a_k\cos kx+b_k\sin kx\right) \end{align}SN​(x)=2a0​​+k=1∑N​(ak​coskx+bk​sinkx)​​
(2)

と書く．フーリエ級数の収束を考えるとは，NNN を大きくしたときに SN(x)S_N(x)SN​(x) がどの値へ近づくかを考えることである．

連続な点では，よい条件のもとで SN(x)S_N(x)SN​(x) は f(x)f(x)f(x) に近づく．一方，不連続な点では，SN(x)S_N(x)SN​(x) はその点で定義された関数値そのものではなく，左右から近づいた値の平均に近づく．

ディリクレ・ジョルダンの定理¶

フーリエ級数の収束先を知るための代表的な定理として，ディリクレ・ジョルダンの定理がある．

ディリクレ・ジョルダンの定理

関数 f(x)f(x)f(x) が周期 2π2\pi2π の周期関数であり，区間 [−π,π][-\pi,\pi][−π,π] 上で有界変動であるとする．このとき，f(x)f(x)f(x) のフーリエ級数の部分和 SN(x)S_N(x)SN​(x) は，各点 xxx で

lim⁡N→∞SN(x)=f(x+0)+f(x−0)2\lim_{N\to\infty}S_N(x)=\frac{f(x+0)+f(x-0)}{2}N→∞lim​SN​(x)=2f(x+0)+f(x−0)​
(3)

へ収束する．ここで，f(x+0)f(x+0)f(x+0) は右極限，f(x−0)f(x-0)f(x−0) は左極限を表す．

特に，f(x)f(x)f(x) が点 xxx で連続であれば f(x+0)=f(x−0)=f(x)f(x+0)=f(x-0)=f(x)f(x+0)=f(x−0)=f(x) なので，フーリエ級数は f(x)f(x)f(x) に収束する．一方，不連続点では左右極限の平均に収束する．この性質は，フーリエ級数で段差を近似するときに非常に重要である．

有界変動という条件は少し抽象的だが，有限個の点を除いて滑らかで，有限個の不連続点しか持たないような段差のある関数を考えている．

Pythonによる部分和の可視化¶

では，部分和 SN(x)S_N(x)SN​(x) を SymPy で計算して，項数を増やしたときの近似の様子を確認する．まず，ステップ関数を Piecewise で定義し，fourier_series でフーリエ級数を求める．その後，truncate() で有限個の項に打ち切って可視化する．

from sympy import symbols, pi, Piecewise, fourier_series
from sympy.plotting import plot

x = symbols("x")
f = Piecewise((1, x <= 0), (0, True))

ff = fourier_series(f, (x, -pi, pi))

項数を増やすと，不連続点から離れた場所では近似がよくなる．一方，不連続点の近くでは振動が残り，段差の上側や下側にはみ出す部分が現れる．この現象を ギブス現象 という．ギブス現象は「項数が少ないために起こる単なる誤差」ではなく，不連続をフーリエ級数で表すときに本質的に現れる現象である．
